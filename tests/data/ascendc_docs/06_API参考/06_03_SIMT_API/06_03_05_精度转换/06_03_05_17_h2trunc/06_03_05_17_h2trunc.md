功能说明

获取对输入数据各元素的浮点数截断后的整数。

函数原型

```cpp
inline half2 h2trunc(half2 x)inline bfloat16x2_t h2trunc(bfloat16x2_t x)
```

参数说明

表6-1593参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据各元素的浮点数截断后的整数。特别场景说明如下：

●当输入元素为nan时，返回值为nan。

●当输入元素为inf时，返回值为inf。

●当输入元素为-inf时，返回值为-inf。

约束说明

SIMT编程场景当前不支持使用该接口。

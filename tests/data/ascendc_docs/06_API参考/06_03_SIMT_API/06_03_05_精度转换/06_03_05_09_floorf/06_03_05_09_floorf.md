功能说明

获取对输入数据四舍五入后的整数。

函数原型

```cpp
inline long long int llroundf(float x)
```

参数说明

表6-1584参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

对输入四舍五入后的整数。特别场景说明如下：

●当x为nan时，返回值为0

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

功能说明

获取小于或等于输入数据的最大整数值。

函数原型

```cpp
inline float floorf(float x)
```

参数说明

表6-1585参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

小于或等于输入数据的最大整数值。特别场景说明如下：

●当x为nan时，返回值为nan。

●当x为inf时，返回值为inf。

●当x为-inf时，返回值为-inf。

功能说明

获取小于或等于输入数据的最大整数值。

函数原型

```cpp
inline half hfloor(half x)inline bfloat16_t hfloor(bfloat16_t x)
```

参数说明

表6-1586参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

小于或等于输入数据的最大整数值。特别场景说明如下：

●当x为nan时，返回值为nan。

●当x为inf时，返回值为inf。

●当x为-inf时，返回值为-inf。

约束说明

SIMT编程场景当前不支持使用该接口。

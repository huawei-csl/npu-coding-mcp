功能说明

获取输入数据x除以y的余数。求余数时，商取x除以y浮点数结果的整数部分。

函数原型

```cpp
inline float fmodf(float x, float y)
```

参数说明

表6-1549参数说明

参数名输入/输出

描述

x输入源操作数。

y输入源操作数。

返回值说明

输入数据x除以y的余数。

●x为0时，y不为0时，返回值为0。

●x为有限数，y为inf或-inf时，返回值为x。

●x为inf或-inf，或y为0时，返回值为nan。

●x，y任意一个为nan时，返回值为nan。

约束说明

SIMT编程场景当前不支持使用该接口。

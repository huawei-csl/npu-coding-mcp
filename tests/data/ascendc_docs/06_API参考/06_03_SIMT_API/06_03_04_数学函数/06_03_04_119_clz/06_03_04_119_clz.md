功能说明

从输入数据的二进制最高有效位开始，返回连续的前导零的位数。

函数原型

```cpp
int __clz(int x)int __clz(long long x)int __clz(long x)
```

参数说明

表6-1573参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

返回输入整数二进制表示中连续高位零的数量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/device_functions.h"头文件。

```cpp
#include "simt_api/device_functions.h"
```

表6-1492参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的三角函数正弦值。

●当x为inf时，返回值为nan。

●当x为-inf时，返回值为nan。

●当x为nan时，返回值为nan。

约束说明

使用本接口时，线程配置最大不超过1024，否则有栈溢出风险。

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

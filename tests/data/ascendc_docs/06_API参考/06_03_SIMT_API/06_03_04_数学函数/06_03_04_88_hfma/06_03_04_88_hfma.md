返回值说明

x * y+ z的值。

●x为±inf，y为±0，返回nan。

●x为±0，y为±inf，返回nan。

●x*y为inf，z为-inf，返回nan。

●x*y为-inf，z为inf，返回nan。

●x*y+z超出对应类型范围的最大值，返回inf。

●x*y+z小于对应类型范围的最小值，返回-inf。

●x、y、z任意一个为nan，返回nan。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

功能说明

对输入数据x、y、z，计算x与y相乘加上z的结果。


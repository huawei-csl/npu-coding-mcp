参数名输入/输出描述

x输入源操作数。

返回值说明

以2为底的x的对数，并向下取整后的值。

●当x为inf时，返回值为2147483647。

●当x为-inf时，返回值为2147483647。

●当x为nan时，返回值为-2147483648。

●当x为0时，返回值为-2147483648。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

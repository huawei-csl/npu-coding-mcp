功能说明

获取输入uint64类型数据x和y乘积的高64位。

![p3265_img001.png](../../images/p3265_img001.png)

函数原型

```cpp
unsigned long long __umul64hi(unsigned long long x, unsigned long long y)
```

参数说明

表6-1570参数说明

参数名输入/输出

描述

x输入源操作数，乘数。

y输入源操作数，乘数。

返回值说明

输入uint64类型数据x和y乘积的高64位。

约束说明

SIMT编程场景当前不支持使用该接口。

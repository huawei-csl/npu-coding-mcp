函数原型

```cpp
inline float sinhf(float x)
```

参数说明

表6-1495参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的双曲正弦值。

●当x为inf时，返回值为inf。

●当x为-inf时，返回值为-inf。

●当x为nan时，返回值为nan。

●当x为0时，返回值为0。

约束说明

SIMT编程场景当前不支持使用该接口。

本接口支持的输入数据为[-89.4, 89.4]范围内的数据。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

功能说明

获取输入数据与π相乘的正弦值。

![p3161_img001.png](../../images/p3161_img001.png)

函数原型

```cpp
inline float sinpif(float x)
```

参数说明

表6-1496参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据与π相乘的正弦值。

●当x*π超出float最大范围，返回值为nan。

●当x*π超出float最小范围，返回值为nan。

●当x为inf时，返回值为nan。

●当x为-inf时，返回值为nan。

●当x为nan时，返回值为nan。

约束说明

使用本接口时，线程配置最大不超过1024，否则有栈溢出风险。

SIMT编程场景当前不支持使用该接口。

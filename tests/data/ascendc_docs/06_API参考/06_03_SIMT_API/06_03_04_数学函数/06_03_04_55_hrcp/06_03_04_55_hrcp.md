●当输入元素为nan时，返回值为nan。

约束说明

本接口支持的输入数据范围为x大于等于0，否则返回值为nan。

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用half2类型接口需要包含"simt_api/asc_fp16.h"头文件，使用bfloat16x2_t类型接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"#include "simt_api/asc_bf16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelRsqrt(__gm__ half2* dst, __gm__ half2* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = h2rsqrt(x[idx]);}

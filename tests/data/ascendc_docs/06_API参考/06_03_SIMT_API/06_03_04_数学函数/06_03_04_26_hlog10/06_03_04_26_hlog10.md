功能说明


参数说明

表6-1480参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

以10为底的x的对数。

●当x小于0或x为nan时，返回值为nan。

●当x为0时，返回值为-inf。

●当x为inf时，返回值为inf。

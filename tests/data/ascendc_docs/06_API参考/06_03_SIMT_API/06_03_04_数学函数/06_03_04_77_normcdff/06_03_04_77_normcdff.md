功能说明

获取输入数据x的标准正态分布的累积分布函数值。

![p3211_img001.png](../../images/p3211_img001.png)

函数原型

```cpp
inline float normcdff(float x)
```

参数说明

表6-1531参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的标准正态分布的累积分布函数值。

●当x为-inf，返回值为0。

●当x为inf，返回值为1。

●当x为nan，返回值为nan。

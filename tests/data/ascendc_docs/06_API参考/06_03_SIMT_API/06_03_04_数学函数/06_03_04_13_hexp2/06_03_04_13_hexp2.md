参数名输入/输出描述

x输入源操作数。

返回值说明

2的x次方。

●当x为inf时，返回值为inf。

●当x为-inf时，返回值为0。

●当x为nan时，返回值为nan。

●当结果超出float的最大范围时，返回值为inf。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

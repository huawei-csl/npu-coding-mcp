功能说明

将输入参数分解为小数部分和整数部分。

函数原型

```cpp
inline float modff(float x, float *n)
```

参数说明

表6-1556参数说明

参数名输入/输出

描述

x输入源操作数。

n输出Unified Buffer、Global Memory或栈空间的地址，用于存储输入数据的的整数部分。

返回值说明

输入参数的小数部分。

●当x为nan时，返回值为nan。

●当x为inf时，返回值为0。

●当x为-inf时，返回值为0。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

参数说明

表6-1472参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

e的x次方减1。

●当x为inf时，返回值为inf。

●当x为-inf时，返回值为-1.0。

●当x为nan时，返回值为nan。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelExpm1(__gm__ float* dst, __gm__ float* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = expm1f(x[idx]);}

功能说明

获取以e为底，输入数据的对数。

![p3130_img001.png](../../images/p3130_img001.png)

函数原型

```cpp
inline float logf(float x)
```

参数说明

表6-1473参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

以e为底的x的对数。

●当x小于0或x为nan时，返回值为nan。

●当x为0时，返回值为-inf。

●当x为inf时，返回值为inf。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelLog(__gm__ float* dst, __gm__ float* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = logf(x[idx]);}

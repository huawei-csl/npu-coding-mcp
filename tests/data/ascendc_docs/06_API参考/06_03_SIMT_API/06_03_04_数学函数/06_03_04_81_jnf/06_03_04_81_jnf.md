●当x=0，返回值为0。

●当x=inf或-inf，返回值为0。

●当x=nan，返回值为nan。

约束说明

使用本接口时，配置的线程数不应超过256，否则有栈溢出风险。

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(256) inline void KernelJ1(__gm__ float* dst, __gm__ float* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = j1f(x[idx]);}

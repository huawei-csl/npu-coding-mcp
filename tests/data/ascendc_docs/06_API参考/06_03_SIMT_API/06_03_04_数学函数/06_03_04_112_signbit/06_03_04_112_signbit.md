功能说明

获取输入数据的符号位。

函数原型

```cpp
inline int signbit(float x)
```

参数说明

表6-1566参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

返回输入数据的符号位。

●当x为nan时，返回值为0。

●当x为inf时，返回值为0。

●当x为-inf时，返回值为1。

约束说明

SIMT编程场景当前不支持使用该接口。

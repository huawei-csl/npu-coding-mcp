功能说明

判断是否有活跃线程的输入不为0。

当Warp内所有活跃线程执行本接口后，对所有活跃线程的输入操作数predicate进行判断，所有活跃线程的predicate均为0，返回0，否则返回1。Warp内所有活跃线程返回相同的结果。

函数原型

```cpp
inline int32_t asc_any(int32_t predicate)
```

参数说明

表6-1622参数说明

参数名输入/输出

描述

predicate

输入操作数。

返回值说明

当Warp内所有活跃线程的输入均为0，返回0，否则返回1。

无

返回值说明

由四个unsigned long int类型数字构成的新ulong4向量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/vector_functions.h"头文件。

```cpp
#include "simt_api/vector_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel_make_ulong4(__gm__ ulong4* dst, __gm__ unsigned long int* x, __gm__ unsigned long int* y, __gm__ unsigned long int* z, __gm__ unsigned long int* w){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = make_ulong4(x[idx], y[idx], z[idx], w[idx]);}

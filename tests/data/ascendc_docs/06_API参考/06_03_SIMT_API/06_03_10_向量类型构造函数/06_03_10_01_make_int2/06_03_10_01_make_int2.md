功能说明

将unsigned short int的数据按位重新解释为bfloat16，即将unsigned short int的数据存储的位按照bfloat16的格式进行读取。

函数原型

```cpp
inline bfloat16_t __ushort_as_bfloat16(const unsigned short int x)
```

参数说明

表6-1886参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

unsigned short int的数据按位重新解释为bfloat16的值。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel__ushort_as_bfloat16(__gm__ bfloat16_t* dst, __gm__ unsigned short int* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = __ushort_as_bfloat16(x[idx]);}

功能说明

使用给定的两个int类型的数据创建一个int2类型的向量。

函数原型

```cpp
inline int2 make_int2(int x, int y)
```

参数说明

表6-1887参数说明

参数名输入/输出

描述

x输入源操作数。

y输入源操作数。

返回值说明

由两个int类型数字构成的新int2向量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/vector_functions.h"头文件。

```cpp
#include "simt_api/vector_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel_make_int2(__gm__ int2* dst, __gm__ int* x, __gm__ int* y)

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"
```

调用示例

SIMD与SIMT混合编程场景：// 使用小向量可提升数据搬运效率__simt_vf__ __launch_bounds__(1024) inline void simt_low2bfloat162(__gm__ bfloat16x2_t* input, __gm__ bfloat16x2_t* output, uint32_t input_total_length){    uint32_t idx = blockIdx.x * blockDim.x + threadIdx.x;    // 每个线程处理1个bfloat16x2_t类型的数据，即2个bfloat16_t类型的数据，因此idx >= input_total_length / 2的线程不处理数据    if (idx > input_total_length / 2) {        return;    }    output[idx] = __low2bfloat162(input[idx]);}__global__ __vector__ void cast_kernel(__gm__ bfloat16_t* input, __gm__ bfloat16_t* output, uint32_t input_total_length){    asc_vf_call<simt_low2bfloat162>(dim3(1024),  (__gm__ bfloat16x2_t*)input, (__gm__ bfloat16x2_t*)output, input_total_length);}

功能说明

将输入数据的低16位转换为浮点数并返回结果。

<!-- page 3641 -->

函数原型

```cpp
inline float __low2float(const half2 x)__simt_callee__ inline float __low2float(const bfloat16x2_t x)
```

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用bfloat16x2_t接口需要包含"simt_api/asc_bf16.h"头文件，使用half2接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"#include "simt_api/asc_fp16.h"
```

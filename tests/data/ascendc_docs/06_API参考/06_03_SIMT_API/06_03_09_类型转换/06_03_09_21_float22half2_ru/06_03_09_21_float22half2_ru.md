功能说明

将float2类型数据的两个分量遵循CAST_CEIL模式转换为半精度浮点数，返回转换后的half2类型数据。

函数原型

```cpp
inline half2 __float22half2_ru(const float2 x)
```

参数说明

表6-1656参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

将输入的两个分量遵循CAST_CEIL模式转换成的half2类型数据。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"
```

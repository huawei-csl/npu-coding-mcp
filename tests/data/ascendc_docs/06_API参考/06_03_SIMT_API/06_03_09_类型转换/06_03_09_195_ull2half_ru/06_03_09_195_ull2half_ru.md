功能说明

遵循CAST_CEIL模式，将uint64类型数据转换为half类型数据，返回转换后的值。

函数原型

```cpp
inline half __ull2half_ru(const unsigned long long int x)
```

约束说明

SIMT编程场景当前不支持使用该接口。

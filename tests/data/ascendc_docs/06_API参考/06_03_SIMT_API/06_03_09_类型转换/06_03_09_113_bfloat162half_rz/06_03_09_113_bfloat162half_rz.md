约束说明

使用此接口前需将CTRL[60]寄存器设置为0，否则饱和模式不生效。设置方式请参见控制饱和行为的方式。

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel__bfloat162half_rn_sat(__gm__ half* dst, __gm__ bfloat16_t* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = __bfloat162half_rn_sat(x[idx]);}

功能说明

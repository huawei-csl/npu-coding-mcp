功能说明

将hifloat8x2_t类型数据的两个分量转换为浮点数，返回转换后的float2类型数据。

函数原型

```cpp
inline float2 __hif822float2(const hifloat8x2_t x)
```

参数说明

表6-1852参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

将hifloat8x2_t类型输入转换成的float2类型数据。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp8.h"头文件。

```cpp
#include "simt_api/asc_fp8.h"
```

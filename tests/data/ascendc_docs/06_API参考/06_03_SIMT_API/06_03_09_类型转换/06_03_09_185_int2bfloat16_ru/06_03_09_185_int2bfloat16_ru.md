约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel__int2bfloat16_rd(__gm__ bfloat16_t* dst, __gm__ int32_t* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = __int2bfloat16_rd(x[idx]);}

功能说明

遵循CAST_CEIL模式，将int32类型数据转换为bfloat16类型数据，返回转换后的值。

函数原型

```cpp
inline bfloat16_t __int2bfloat16_ru(const int x)
```

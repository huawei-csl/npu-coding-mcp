返回值说明

输入遵循CAST_ROUND模式转换成的half类型数据。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel__ull2half_rna(__gm__ half* dst, __gm__ uint64_t* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = __ull2half_rna(x[idx]);}

功能说明

遵循CAST_RINT模式，将uint64类型数据转换为bfloat16类型数据，返回转换后的值。

函数原型

```cpp
inline bfloat16_t __ull2bfloat16_rn(const unsigned long long int x)
```

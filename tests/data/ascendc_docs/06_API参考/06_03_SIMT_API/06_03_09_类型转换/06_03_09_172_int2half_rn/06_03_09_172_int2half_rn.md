功能说明

遵循CAST_ROUND模式，将int32类型数据转换为浮点数，返回转换后的值。

函数原型

```cpp
inline float __int2float_rna(const int x)
```

参数说明

表6-1806参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入遵循CAST_ROUND模式转换成的浮点数。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/device_functions.h"头文件。

```cpp
#include "simt_api/device_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel__int2float_rna(__gm__ float* dst, __gm__ int32_t* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = __int2float_rna(x[idx]);}

功能说明

将bfloat16_t数据遵循CAST_TRUNC模式取整。

函数原型

```cpp
inline bfloat16_t __bfloat162bfloat16_rz(const bfloat16_t x)
```

参数说明

表6-1758参数说明

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

```cpp
#include "simt_api/asc_bf16.h"
```

## 6.3.9 类型转换

## 6.3.9.1 简介

类型转换包括隐式转换和显式转换。隐式转换在类型不匹配时由系统自动执行。显式转换则通过调用类型转换API来主动完成，API会将输入数据从源类型转换为目标类型，并返回转换结果。整个转换过程遵循规定的处理规则。类型转换的结果受舍入模式和饱和模式的影响。关于浮点数的表示方式和舍入规则，请参见类型转换的相关说明。

隐式转换

隐式转换是指在不显式指定目标类型的情况下，系统自动将数据从源类型转换为目标类型的行为。

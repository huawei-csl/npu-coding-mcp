uint64_t mask = 128;half scalar = 0.001;// repeatTime = 4, 单次迭代处理128个数，计算512个数需要迭代4次// dstBlkStride, srcBlkStride = 1, 每个迭代内src0参与计算的数据地址间隔为1个datablock，表示单次迭代内数据连续读取和写入// dstRepStride, srcRepStride = 8, 相邻迭代间的地址间隔为8个datablock，表示相邻迭代间数据连续读取和写入AscendC::LeakyRelu(dstLocal, srcLocal, scalar, mask, 4, {1, 1, 8, 8});

●tensor高维切分计算样例-mask逐bit模式uint64_t mask[2] = { UINT64_MAX, UINT64_MAX };half scalar = 0.001;// repeatTime = 4, 单次迭代处理128个数，计算512个数需要迭代4次// dstBlkStride, srcBlkStride = 1, 每个迭代内src0参与计算的数据地址间隔为1个datablock，表示单次迭代内数据连续读取和写入// dstRepStride, srcRepStride = 8, 相邻迭代间的地址间隔为8个datablock，表示相邻迭代间数据连续读取和写入AscendC::LeakyRelu(dstLocal, srcLocal, scalar, mask, 4, {1, 1, 8, 8});

●tensor前n个数据计算样例half scalar = 0.001;// 算子输入的数据类型为half, 需要参与计算的元素个数为512AscendC::LeakyRelu(dstLocal, srcLocal, scalar, 512);

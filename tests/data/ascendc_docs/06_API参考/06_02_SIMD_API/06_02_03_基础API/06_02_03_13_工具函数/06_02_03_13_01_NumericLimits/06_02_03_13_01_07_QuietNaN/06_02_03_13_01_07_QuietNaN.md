功能说明

返回指定数据类型的负无穷大值。

Atlas 350 加速卡，支持的数据类型为：half/bfloat16_t/float。

函数原型

●标量接口，返回值为标量constexpr __aicore__ static inline T NegativeInfinity()

●矢量接口，为dstLocal前count个元素赋负无穷大值__aicore__ static inline void NegativeInfinity(const LocalTensor<T>& dstLocal, uint32_t count)

参数说明

表6-809参数说明

参数名输入/输出

描述

dstLocal输出目的操作数。

类型为LocalTensor，支持的TPosition为VECIN/VECCALC/VECOUT。

LocalTensor的起始地址需要32字节对齐。

count输入输入数据元素个数。

返回值说明

标量接口返回值为对应数据类型的负无穷大值。

矢量接口无返回值。

约束说明

无。

调用示例

●标量接口float value = AscendC::NumericLimits<float>::NegativeInfinity();

●矢量接口AscendC::NumericLimits<float>::NegativeInfinity(dstLocal, 256);// 返回256个float类型的负无穷大值

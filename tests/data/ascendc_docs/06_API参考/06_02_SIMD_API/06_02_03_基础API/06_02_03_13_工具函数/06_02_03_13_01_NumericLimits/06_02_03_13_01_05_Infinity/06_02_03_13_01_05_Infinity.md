功能说明

返回指定数据类型的正无穷大值。

Atlas 350 加速卡，支持的数据类型为：half/bfloat16_t/float。

函数原型

●标量接口，返回值为标量constexpr __aicore__ static inline T Infinity()

●矢量接口，为dstLocal前count个元素赋正无穷大值__aicore__ static inline void Infinity(const LocalTensor<T>& dstLocal, uint32_t count)

功能说明

返回指定数据类型的最大有限值。

Atlas 350 加速卡，支持的数据类型为：uint8_t/int8_t/uint16_t/int16_t/half/bfloat16_t/uint32_t/int32_t/float。

函数原型

●标量接口，返回值为标量constexpr __aicore__ static inline T Max()

●矢量接口，为dst前count个元素赋最大有限值__aicore__ static inline void Max(const LocalTensor<T>& dst, uint32_t count)

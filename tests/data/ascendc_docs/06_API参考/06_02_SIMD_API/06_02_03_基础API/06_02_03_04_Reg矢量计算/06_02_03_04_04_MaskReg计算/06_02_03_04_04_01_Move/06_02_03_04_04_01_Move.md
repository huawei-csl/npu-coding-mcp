功能说明

将src中的元素复制到dst中的对应位置。如果有输入mask，则仅复制被mask选定的有效元素，无效元素填0。

函数原型

●有输入mask__simd_callee__ inline void Move(MaskReg& dst, MaskReg& src, MaskReg& mask)

●无输入mask__simd_callee__ inline void Move(MaskReg& dst, MaskReg& src)

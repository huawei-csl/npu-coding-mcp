功能说明

将源操作数的浮点数元素截断到整数位，同时源操作数的数据类型保持不变。

函数原型

```cpp
template <typename T = DefaultType, RoundMode roundMode = RoundMode::CAST_NONE, MaskMergeMode mode = MaskMergeMode::ZEROING, typename S>__simd_callee__ inline void Truncate(S& dstReg, S& srcReg, MaskReg& mask)
```

参数说明

表6-599模板参数说明

参数名描述

T操作数数据类型。

Atlas 350 加速卡，dstReg/srcReg支持的数据类型为：half、float、bfloat16_t。

roundMode舍入模式，取值如下：

●RoundMode::CAST_NONE：默认值，在转换会导致精度丢失的场景下相当于CAST_RINT，其他场景下无效果；

●RoundMode::CAST_RINT：返回最接近参数的整数，如果有2个整数同样接近，则会返回偶数的那个；

●RoundMode::CAST_ROUND：round模式，四舍五入求整；

●RoundMode::CAST_FLOOR：floor模式，向下取整；

●RoundMode::CAST_CEIL：ceil模式，向上取整；

●RoundMode::CAST_TRUNC：truncation模式，向0取整，舍弃小数位。

<!-- page 1675 -->

参数名描述

mode选择MERGING模式或ZEROING模式。

●ZEROING，mask未筛选的元素在dst中置零

●MERGING，当前不支持。

SsrcReg/dstReg 类型，例如RegTensor<float>，由编译器自动推导，用户不需要填写。

表6-600参数说明

参数名输入/输出

描述

dstReg输出目的操作数。

类型为RegTensor。

srcReg输入源操作数。

类型为RegTensor。

源操作数的数据类型需要与目的操作数保持一致。

mask输入源操作数元素操作的有效指示，详细说明请参考MaskReg。

返回值

无

约束说明

●非饱和模式下，输入nan，输出nan; 输入值超出输入类型的最值范围，则输出+/-inf。

●饱和模式下，输入nan, 输出为0，输入值超出输入类型的最值范围，则饱和为最值。

●float类型只支持不饱和模式。

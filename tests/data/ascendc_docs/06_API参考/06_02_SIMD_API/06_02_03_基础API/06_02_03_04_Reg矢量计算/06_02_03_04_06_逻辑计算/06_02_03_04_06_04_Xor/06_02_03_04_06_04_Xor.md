功能说明

本节介绍两种接口，分别用于对RegTensor和MaskReg进行操作：

●对RegTensor操作：

根据mask对输入数据srcReg0、srcReg1按元素异或（^）操作，将结果写入dstReg。计算公式如下：

●对MaskReg操作：

将两个输入MaskReg的有效bit进行逻辑异或运算得到新的MaskReg。

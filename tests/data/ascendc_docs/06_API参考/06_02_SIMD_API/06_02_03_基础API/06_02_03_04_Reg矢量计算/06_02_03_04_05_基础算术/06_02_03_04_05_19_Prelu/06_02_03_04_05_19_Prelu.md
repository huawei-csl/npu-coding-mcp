功能说明

源操作数src0大于0的情况下直接将src0写入目的操作数dst，否则将源操作数src0*src1的结果写入dst。计算公式如下：

函数原型

```cpp
template <typename T = DefaultType, MaskMergeMode mode = MaskMergeMode::ZEROING, typename U>__simd_callee__ inline void Prelu(U &dstReg, U &srcReg0, U &srcReg1, MaskReg &mask);
```

参数说明

表6-536模板参数说明

参数名描述

T矢量目的操作数和源操作数的数据类型。

Atlas 350 加速卡，支持的数据类型为：half、float。

mode●选择MERGING模式或ZEROING模式。

–ZEROING, mask未筛选的元素在dst中置零。

–MERGING, 当前不支持。

U目的操作数的RegTensor类型，例如RegTensor<half>，由编译器自动推导，用户不需要填写。

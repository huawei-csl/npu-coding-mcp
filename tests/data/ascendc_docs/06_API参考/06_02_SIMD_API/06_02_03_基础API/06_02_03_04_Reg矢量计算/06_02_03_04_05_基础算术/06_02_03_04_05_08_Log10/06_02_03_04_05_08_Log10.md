功能说明

根据mask对输入数据srcReg进行求以10为底的对数操作，将结果写入dstReg。计算公式如下：

函数原型

```cpp
template <typename T = DefaultType, auto mode = MaskMergeMode::ZEROING, typename U>__simd_callee__ inline void Log10(U& dstReg, U& srcReg, MaskReg& mask)
```

参数说明

表6-512模板参数说明

参数名描述

T操作数数据类型。

Atlas 350 加速卡，支持的数据类型为：half、float。

mode可配置为MaskMergeMode的枚举或Log10SpecificMode的结构体。

●MaskMergeMode，选择MERGING模式或ZEROING模式。

–ZEROING，mask未筛选的元素在dstReg中置零。

–MERGING，暂不支持。

●Log10SpecificMode，定义如下：enum class Log10Algo {    INTRINSIC = 0,    PRECISION_1ULP_FTZ_TRUE,    PRECISION_1ULP_FTZ_FALSE,};struct Log10SpecificMode{    MaskMergeMode mrgMode = MaskMergeMode::ZEROING,    Log10Algo algo = Log10Algo::INTRINSIC;};

–mrgMode：选择MERGING模式或ZEROING模式。

–algo：用于配置Subnormal模式。

–Log10Algo::INTRINSIC、Log10Algo::PRECISION_1ULP_FTZ_TRUE，使用单指令计算得出结果，所有Subnormal被近似为0。

–Log10Algo::PRECISION_1ULP_FTZ_FALSE，支持Subnormal数据计算。

U目的操作数的RegTensor类型，例如RegTensor<half>，由编译器自动推导，用户不需要填写。

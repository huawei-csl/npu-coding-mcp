功能说明

根据mask对输入数据srcReg0、srcReg1按元素相乘操作，将结果写入dstReg。计算公式如下：

函数原型

```cpp
template <typename T = DefaultType, MaskMergeMode mode = MaskMergeMode::ZEROING, typename U>__simd_callee__ inline void Mul(U& dstReg, U& srcReg0, U& srcReg1, MaskReg& mask)
```

参数说明

表6-521模板参数说明

参数名描述

T操作数数据类型。

Atlas 350 加速卡，支持的数据类型为：uint16_t/int16_t/uint32_t/int32_t/half/float/bfloat16_t/uint64_t/int64_t/complex32/complex64

mode选择MERGING模式或ZEROING模式。

●ZEROING, mask未筛选的元素在dstReg中置零。

●MERGING, 当前不支持。

U目的操作数的RegTensor类型，例如RegTensor<half>，由编译器自动推导，用户不需要填写。

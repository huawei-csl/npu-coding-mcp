功能说明

以传入的scalar的值为起始值，生成递增/递减的索引，并将产生的索引保存在dstReg中。

函数原型

```cpp
template <typename T = DefaultType, IndexOrder order = IndexOrder::INCREASE_ORDER, typename U, typename S>__simd_callee__ inline void Arange(S& dstReg, U scalarValue);
```

功能说明

逐元素比较两个Tensor大小，如果比较后的结果为真，则输出结果的对应比特位为1，否则为0。

函数原型

```cpp
template <typename T = DefaultType, CMPMODE mode = CMPMODE::EQ, typename U>__simd_callee__ inline void Compare(MaskReg& dst, U& srcReg0, U& srcReg1, MaskReg& mask)
```

参数说明

表6-581模板参数说明

参数名描述

T操作数数据类型。

Atlas 350 加速卡，支持的数据类型为：uint8_t/int8_t/uint16_t/int16_t/uint32_t/int32_t/half/float/bfloat16_t/uint64_t/int64_t

<!-- page 1639 -->

参数名描述

mode比较模式。支持如下取值：

●LT：小于（less than）

●GT：大于（greater than）

●GE：大于或等于（greater than or equal to）

●EQ：等于（equal to）

●NE：不等于（not equal to）

●LE：小于或等于（less than or equal to）

U源操作数和目的操作数的RegTensor类型，例如RegTensor<half>，由编译器自动推导，用户不需要填写。

表6-582参数说明

参数名输入/输出

描述

dst输出MaskReg类型，目的操作数。

srcReg0、srcReg1

输入源操作数。

类型为RegTensor。

两个源操作数的数据类型需要与目的操作数保持一致。

mask输入源操作数元素操作的有效指示，详细说明请参考MaskReg。

返回值说明

无

约束说明

srcReg0和srcReg1可以是同一个RegTensor。

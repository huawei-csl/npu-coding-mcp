功能说明

将传入的数据打包成Shape数据结构。

函数原型

```cpp
template <typename... Ts>__aicore__ inline constexpr Shape<Ts...> MakeShape(const Ts&... t)
```

参数说明

参数名输入/输出描述

Ts...输入表示输入类型的形参包，使用方法和约束说明同Std::tuple。

返回值说明

Shape结构类型（Std::tuple类型的别名），用于定义数据的逻辑形状，例如二维矩阵的行数和列数或多维张量的各维度大小。定义如下:

```cpp
template <typename... Shapes>using Shape = Std::tuple<Shapes...>;
```

约束说明

同Std::tuple。

功能说明

在不改变Tiling的情况下，重新设置本次计算的singleCoreM/singleCoreN/singleCoreK，以元素为单位。

函数原型

```cpp
__aicore__ inline void SetTail(int tailM = -1, int tailN = -1, int tailK = -1)
```

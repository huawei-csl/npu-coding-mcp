参数名含义

Tps...Tps...为传入tuple的模板参数包，代表传递给make_tuple的参数类型，参数个数范围为[0, 64]。

Atlas 350 加速卡，支持的数据类型为：bool、int4b_t、int8_t、uint8_t、fp8_e8m0_t、int16_t、uint16_t、half、bfloat16_t、int32_t、uint32_t、float、int64_t、uint64_t、LocalTensor、GlobalTensor。

Atlas A3 训练系列产品/Atlas A3 推理系列产品，支持的数据类型为：bool、int4b_t、int8_t、uint8_t、int16_t、uint16_t、half、bfloat16_t、int32_t、uint32_t、float、int64_t、uint64_t、LocalTensor、GlobalTensor。

Atlas A2 训练系列产品/Atlas A2 推理系列产品，支持的数据类型为：bool、int4b_t、int8_t、uint8_t、int16_t、uint16_t、half、bfloat16_t、int32_t、uint32_t、float、int64_t、uint64_t、LocalTensor、GlobalTensor。

argsargs...是函数参数包，代表传递给make_tuple的实际参数，参数个数范围为[0, 64]。

约束说明

●tuple实例化深度为64，即支持64个元素以内的基础数据类型的聚合。

●make_tuple中需要特别指定数据类型的元素，必须对该元素的数据类型增加强转，否则由编译器自行推断，可能与预期不符。

●不支持数组等可变长度的数据类型。

●不支持隐式转换构造函数。

返回值说明

一个tuple对象，此对象包含传递的参数副本。

调用示例

```cpp
AscendC::Std::tuple<uint32_t, float, bool> test = AscendC::Std::make_tuple(22, 3.3, true);
```

更多调用示例请参见示例。

## 6.4.1.4 类型特性

type_traits是Ascend C标准库中的一个头文件（<type_traits>），它提供了一系列用于在编译时进行类型检查、类型转换和类型操作的模板元编程工具。这些工具可以帮助开发者在程序编译时进行类型相关的决策，从而提高代码的安全性和性能。

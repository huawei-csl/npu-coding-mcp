参数名含义

U输入数据src1的数据类型。当前支持的数据类型为bool、int8_t、uint8_t、int16_t、uint16_t、int32_t、uint32_t、float、int64_t、uint64_t。

预留类型，当前必须与T保持一致。

表6-1917接口参数说明

参数名输入/输出

含义

src0输入源操作数。参与比较的输入。

src1输入源操作数。参与比较的输入。

约束说明

两个源操作数的数据类型必须相同。

返回值说明

两个输入数据中的最大值。

调用示例

```cpp
int64_t src0 = 1;int64_t src1 = 2;int64_t result = AscendC::Std::max(src0, src1); // result: 2
```

约束说明

get函数仅支持const和constexpr常量索引，索引的取值范围为[0, 64)。

返回值说明

调用示例


功能说明

make_tuple是一个实用的函数模板，其作用在于便捷地创建tuple对象。它可以自动推断元素的类型，使代码更简洁，也可以构造元素列表。

函数原型

```cpp
template <typename ...Tps>__aicore__ inline constexpr tuple<unwrap_decay_t<Tps>...> make_tuple(Tps&& ...args)
```

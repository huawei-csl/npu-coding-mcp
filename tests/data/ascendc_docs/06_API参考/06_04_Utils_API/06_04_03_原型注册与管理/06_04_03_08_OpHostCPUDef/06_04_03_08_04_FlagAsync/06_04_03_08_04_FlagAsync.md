## 6.4.3.8.4 FlagAsync

功能说明

此参数为预留参数，请保持固定值“False”。

函数原型

```cpp
OpHostCPUDef &FlagAsync(bool flag)
```

参数说明

参数输入/输出说明

flag输入此参数为预留参数，请保持固定值“False”。

OpHostCPUDef算子定义，OpHostCPUDef请参考6.4.3.8 OpHostCPUDef。

## 6.4.3.8.5 OpKernelLib

功能说明

配置算子调用的kernelLib，HostCPU自定义算子调用的kernelLib固定为“CUSTHOSTCPUKernel”。

函数原型

```cpp
OpHostCPUDef &OpkernelLib(const char *value)
```

参数说明

参数输入/输出说明

value输入kernelLib值，算子调用的kernelLib固定为“CUSTHOSTCPUKernel。

返回值说明

OpHostCPUDef算子定义，OpHostCPUDef请参考6.4.3.8 OpHostCPUDef。

## 6.4.3.9.2 Input

功能说明

某些场景下，同一个算子在不同的AI处理器型号上，其支持的原型输入不同。

通过该接口，可针对不同的AI处理器型号注册差异化的算子输入。调用该接口后会返回一个OpParamDef结构，后续可通过该结构配置算子输入信息。

函数原型

```cpp
OpParamDef &Input(const char *name)
```

参数说明

参数输入/输出说明

name输入算子输入名称。

## 6.4.3.9.3 Output

功能说明

某些场景下，同一个算子在不同的AI处理器型号上，其支持的原型输出不同。

通过该接口，可针对不同的AI处理器型号注册差异化的算子输出。调用该接口后会返回一个OpParamDef结构，后续可通过该结构配置算子输出信息。

函数原型

```cpp
OpParamDef &Output(const char *name)
```

参数说明

参数输入/输出说明

name输入算子输出名称。

返回值说明

算子参数定义，OpParamDef请参考6.4.3.4 OpParamDef。

约束说明

无

## 6.4.5.2.5 Outputs

功能说明

将void* 指针的vector设置为KernelContext的output。

函数原型

```cpp
ContextBuilder &Outputs(std::vector<void *> outputs)
```

参数说明

参数输入/输出说明

outputs输入保存输出的void*指针vector

返回值说明

当前ContextBuilder的对象

约束说明

无

调用示例

```cpp
PlatformInfo platformInfo;auto contextBuilder = context_ascendc::ContextBuilder().Outputs({nullptr, reinterpret_cast<void *>(&platformInfo)});
```

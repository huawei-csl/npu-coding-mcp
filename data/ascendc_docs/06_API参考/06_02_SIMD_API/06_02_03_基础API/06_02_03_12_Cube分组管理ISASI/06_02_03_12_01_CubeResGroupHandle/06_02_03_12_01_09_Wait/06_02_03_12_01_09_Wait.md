功能说明

在调用PostMessage或PostFakeMessage后，查询该消息是否已被AIC处理完。

函数原型

```cpp
template <bool sync = true>__aicore__ inline bool Wait(uint16_t offset)
```

参数说明

表6-795模板参数说明

参数说明

sync查询消息时，程序的运行是否需要等待。参数取值如下：

●true，必须等到AIC处理完该消息后，程序才可以继续运行。

●false，仅查询AIC是否处理完该消息。

表6-538模板参数说明

参数名描述

T矢量目的操作数和源操作数的数据类型。

Atlas 350 加速卡，支持的数据类型为：int8_t/uint8_t/int16_t/uint16_t/half/bfloat16_t/int32_t/uint32_t/float/complex32/int64_t/uint64_t/complex64

U标量源操作数的数据类型。

Atlas 350 加速卡，支持的数据类型为：int8_t/uint8_t/int16_t/uint16_t/half/bfloat16_t/int32_t/uint32_t/float/complex32/int64_t/uint64_t/complex64

mode选择MERGING模式或ZEROING模式。

●ZEROING，mask未筛选的元素在dstReg中置零。

●MERGING，暂不支持。

SdstReg RegTensor类型，例如RegTensor<half>，由编译器自动推导，用户不需要填写。

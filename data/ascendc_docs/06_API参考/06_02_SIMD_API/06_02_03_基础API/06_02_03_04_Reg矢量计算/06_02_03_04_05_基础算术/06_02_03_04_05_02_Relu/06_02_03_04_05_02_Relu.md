功能说明

根据mask对输入数据srcReg进行relu操作，将结果写入dstReg。relu计算公式如下：

![p1556_img001.png](../../../../images/p1556_img001.png)

函数原型

```cpp
template <typename T = DefaultType, MaskMergeMode mode = MaskMergeMode::ZEROING, typename U>__simd_callee__ inline void Relu(U& dstReg, U& srcReg, MaskReg& mask)
```

参数说明

表6-500模板参数说明

参数名描述

T操作数数据类型。

Atlas 350 加速卡，支持的数据类型为：int32_t/int64_t/half/float

mode选择MERGING模式或ZEROING模式。

●ZEROING，mask未筛选的元素在dstReg中置零。

●MERGING，当前不支持。

U目的操作数的RegTensor类型，例如RegTensor<half>，由编译器自动推导，用户不需要填写。

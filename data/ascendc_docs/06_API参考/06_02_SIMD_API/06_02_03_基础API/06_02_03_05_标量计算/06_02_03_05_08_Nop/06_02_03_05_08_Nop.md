功能说明

用户输入延迟的个数，实现延迟若干个cycle。

函数原型

```cpp
template <int count = 1>__aicore__ inline void Nop()
```

参数说明

表6-655参数说明

参数名输入/输出

描述

count输入延迟count个cycle。

约束说明

无

调用示例

如下样例中，延迟10个cycle。

```cpp
Nop<10>();
```

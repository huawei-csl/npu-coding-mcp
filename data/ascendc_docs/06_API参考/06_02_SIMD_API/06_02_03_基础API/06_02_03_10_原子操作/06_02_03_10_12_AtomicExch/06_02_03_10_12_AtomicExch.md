功能说明

在GM内存中执行原子交换操作。具体来说，它读取指定GM地址上的数据，并将新的值存储回同一地址。函数返回旧值。

函数原型

```cpp
template <typename T>__aicore__ inline T AtomicExch(__gm__ T *address, T value)
```

功能说明

返回指定数据类型的发信NaN值（浮点尾数最高位为0）。

Atlas 350 加速卡，支持的数据类型为：half/bfloat16_t/float。

函数原型

●标量接口，返回值为标量constexpr __aicore__ static inline T SignalingNaN()

●矢量接口，为dstLocal前count个元素赋发信NaN值__aicore__ static inline void SignalingNaN(const LocalTensor<T>& dstLocal, uint32_t count)

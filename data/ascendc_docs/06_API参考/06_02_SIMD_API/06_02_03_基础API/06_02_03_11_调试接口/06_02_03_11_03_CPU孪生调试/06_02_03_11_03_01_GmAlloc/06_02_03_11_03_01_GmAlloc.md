功能说明

进行核函数的CPU侧运行验证时，用于创建共享内存：在/tmp目录下创建一个共享文件，并返回该文件的映射指针。

函数原型

```cpp
void *GmAlloc(size_t size)
```

参数说明

参数名输入/输出

描述

size输入用户想要申请的共享内存大小

返回值说明

返回该共享内存空间的首地址。

约束说明

该接口在系统的/tmp目录下生成临时文件，故需要磁盘空间足够才可以正常生成共享内存。

调用示例

uint32_t numBlocks = 8;    // 总核数uint32_t blockLength = 2048;    // 每个核分配的内存大小size_t len = numBlocks * blockLength * sizeof(uint16_t);    // 共享内存大小uint8_t* x = (uint8_t*)AscendC::GmAlloc(len);

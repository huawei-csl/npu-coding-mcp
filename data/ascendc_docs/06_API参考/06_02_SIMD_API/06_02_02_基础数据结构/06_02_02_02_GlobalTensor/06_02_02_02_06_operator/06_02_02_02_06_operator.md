返回值说明

返回PrimType类型的立即数。

约束说明

如果GetValue的Global Memory地址内容存在被外部改写的可能，需要先调用DataCacheCleanAndInvalid，确保Data Cache与Global Memory的Cache一致性，之后再调用此接口。

调用示例

// 数据(GlobalTensor): [0 100 200  ... ]

AscendC::GlobalTensor<uint32_t> tiling_global;tiling_global.SetGlobalBuffer(reinterpret_cast<__gm__ uint32_t *>(tiling), 16);const uint16_t C1 = (uint16_t)tiling_global.GetValue(0);const uint16_t H = (uint16_t)tiling_global(1);  // 示例结果如下：// C1 = 0，H = 100

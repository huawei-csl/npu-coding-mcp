功能说明

获取指定Tensor块的Tag信息，用户可以根据Tag信息对Tensor进行不同操作。

函数原型

```cpp
__aicore__ inline TTagType GetUserTag() const
```

参数说明

无

返回值说明

指定Tensor块的Tag信息。TTagType定义如下：

```cpp
using TTagType = int32_t
```

约束说明

无

调用示例

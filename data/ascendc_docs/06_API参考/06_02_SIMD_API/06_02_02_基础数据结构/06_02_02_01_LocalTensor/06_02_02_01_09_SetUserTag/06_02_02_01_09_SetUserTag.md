功能说明

为Tensor添加用户自定义信息，用户可以根据需要设置对应的Tag。后续可通过GetUserTag获取指定Tensor的Tag信息，并根据Tag信息对Tensor进行相应操作。

函数原型

```cpp
__aicore__ inline void SetUserTag(const TTagType tag)
```

参数说明

表6-58参数说明

参数名输入/输出

描述

tag输入设置的Tag信息，类型TTagType对应为int32_t。

返回值说明

无

约束说明

无

// 示例AscendC::TTagType tag = 10;inputLocal.SetUserTag(tag); // 对LocalTensor设置tag信息。

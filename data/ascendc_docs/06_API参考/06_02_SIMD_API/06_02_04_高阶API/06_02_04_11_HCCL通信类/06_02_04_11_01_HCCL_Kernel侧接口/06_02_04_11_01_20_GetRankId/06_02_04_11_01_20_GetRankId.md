功能说明

获取本卡RankId。该接口默认在所有核上工作，用户也可以在调用前通过GetBlockIdx指定其在某一个核上运行。

函数原型

```cpp
__aicore__ inline uint32_t GetRankId()
```

参数说明

无

返回值说明

返回本卡RankId。

## 6.2.4.11.2.6 SetGroupName

功能说明

设置通信任务所在的通信域。

函数原型

```cpp
uint32_t SetGroupName(const std::string &groupName)
```

参数说明

表6-1376参数说明

参数名输入/输出

描述

groupName

输入当前通信任务所在的通信域。string类型，支持的最大长度为128字节。

返回值说明

●0表示设置成功。

●非0表示设置失败。

约束说明

无

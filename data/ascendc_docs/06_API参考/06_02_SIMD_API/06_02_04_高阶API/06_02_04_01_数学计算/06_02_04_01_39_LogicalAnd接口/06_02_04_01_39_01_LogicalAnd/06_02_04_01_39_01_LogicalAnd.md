maxLivedNodeCount

输出最大存活节点数，表示临时空间是单次计算数据量所占空间的多少倍。

extraBuf输出使用的额外临时空间大小，单位为字节。

返回值说明

无

约束说明

当利用maxLivedNodeCount、extraBuf反推出的currentShapeSize * typeSize < 256B时，currentShapeSize按照256B/typeSize的值向上取整。

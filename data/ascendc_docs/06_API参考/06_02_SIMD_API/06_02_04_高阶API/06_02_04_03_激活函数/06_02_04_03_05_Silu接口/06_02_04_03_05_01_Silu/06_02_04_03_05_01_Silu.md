表6-1156参数列表

接口输入/输出

功能

typeSize输入输入的数据类型大小，单位为字节。比如输入的数据类型为half，此处应传入2。

maxLiveNodeCount

输出最大存活节点数，最大临时空间是输入所占空间的多少倍。

extraBuffer输出使用的额外临时空间大小，单位为byte。

返回值说明

无

约束说明

当利用maxLiveNodeCount，extraBuffer反推出的currentShapeSize * typeSize <256B时，currentShapeSize按照256B/typeSize的值向上取整。

调用示例

```cpp
uint32_t typeSize = sizeof(half);uint32_t maxLiveNodeCount = 0;uint32_t extraBuffer = 0;
```

参数名描述

T操作数的数据类型。

表6-1158接口参数说明

参数名输入/输出

描述

dstLocal输出目的操作数。

类型为LocalTensor，支持的TPosition为VECIN/VECCALC/VECOUT。

srcLocal输入源操作数。

类型为LocalTensor，支持的TPosition为VECIN/VECCALC/VECOUT。

源操作数的数据类型需要与目的操作数保持一致。

dataSize输入实际计算数据元素个数。

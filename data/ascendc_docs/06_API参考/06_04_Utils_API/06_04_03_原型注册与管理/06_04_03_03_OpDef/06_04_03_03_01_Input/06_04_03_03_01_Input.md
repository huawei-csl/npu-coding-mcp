## 6.4.3.3 OpDef

## 6.4.3.3.1 Input

功能说明

注册算子输入，调用该接口后会返回一个OpParamDef结构，后续可通过该结构配置算子输入信息。

函数原型

```cpp
OpParamDef &Input(const char *name)
```

参数说明

参数输入/输出说明

name输入算子输入名称。

返回值说明

算子参数定义，OpParamDef实例，具体请参考6.4.3.4 OpParamDef。

约束说明

参数注册的顺序需要和算子kernel入口函数一致。

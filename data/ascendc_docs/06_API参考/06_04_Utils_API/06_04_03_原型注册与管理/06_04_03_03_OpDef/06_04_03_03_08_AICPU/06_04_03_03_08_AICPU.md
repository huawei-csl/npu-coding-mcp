## 6.4.3.3.8 AICPU

功能说明

设置AI CPU的配置信息：包括算子输入输出的名称、支持的数据类型、数据格式等信息。

函数原型

```cpp
OpAICPUDef &AICPU(void)
```

参数说明

无

返回值说明

OpAICPUDef请参考6.4.3.7 OpAICPUDef。

约束说明

无

## 6.4.3.3.9 HostCPU

功能说明

设置Host CPU的配置信息：包括算子输入输出的name、支持的dtype、format等信息。

函数原型

```cpp
OpHostCPUDef &HostCPU(void)
```

参数说明

无

返回值说明

OpHostCPUDef请参考6.4.3.8 OpHostCPUDef。

约束说明

无

## 6.4.3.3.10 MC2

功能说明

注册该算子是通算融合算子，调用该接口后会返回一个OpMC2Def结构，后续可通过该结构配置通信域名称。

## 6.4.3.4.1 构造&拷贝构造函数

功能说明

构造算子参数的OpParamDef，OpParamDef用于描述算子参数的信息，如数据类型、格式等。一般用户不会直接构造OpParamDef，而是通过Input、Output接口间接构造算子参数的OpParamDef。

函数原型

```cpp
explicit OpParamDef(const char *name);OpParamDef(const OpParamDef &def);
```

参数说明

参数输入/输出说明

name输入算子参数名，同一算子需保证所有参数名各不相同。

def输入已构造完成的算子参数OpParamDef，用于拷贝构造OpParamDef。

返回值说明

无

约束说明

无

## 6.4.3.4.2 ParamType

功能说明

定义算子参数类型。

函数原型

```cpp
OpParamDef &ParamType(Option param_type)
```

参数说明

参数输入/输出说明

**param_type**

输入参数类型，Option取值为：OPTIONAL（可选）、REQUIRED（必选）、DYNAMIC（动态输入）。

返回值说明

OpParamDef算子定义，OpParamDef请参考6.4.3.4 OpParamDef。

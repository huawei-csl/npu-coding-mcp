## 6.4.3.8.6 KernelSo

功能说明

配置HostCPU算子实现文件编译生成的动态库文件的名称。

函数原型

```cpp
OpHostCPUDef &KernelSo(const char *value)
```

参数说明

参数输入/输出说明

value输入HostCPU算子实现文件编译生成的动态库文件的名称。

返回值说明

OpHostCPUDef算子定义，OpHostCPUDef请参考6.4.3.8 OpHostCPUDef。

函数原型

```cpp
OpHostCPUDef &FunctionName(const char *value)
```

参数说明

参数输入/输出说明

value输入算子调用的kernel函数接口名称，functionName固定为“RunCpuKernel”。

返回值说明

OpHostCPUDef算子定义，OpHostCPUDef请参考6.4.3.8 OpHostCPUDef。

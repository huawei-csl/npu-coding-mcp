## 6.4.3.7 OpAICPUDef

## 6.4.3.7.1 Engine

功能说明

配置算子调用的引擎，入参固定为“DNN_VM_AICPU”。

函数原型

```cpp
OpAICPUDef &Engine(const char *value)
```

参数说明

参数输入/输出说明

value输入配置算子调用的引擎名称。

返回值说明

OpAICPUDef算子定义，OpAICPUDef请参考6.4.3.7 OpAICPUDef。

约束说明

无

## 6.4.3.7.2 FlagPartial

功能说明

此参数为预留参数，请保持固定值“False”。

函数原型

```cpp
OpAICPUDef &FlagPartial(bool flag)
```

参数说明

参数输入/输出说明

flag输入此参数为预留参数，请保持固定值“False”。

返回值说明

OpAICPUDef算子定义，OpAICPUDef请参考6.4.3.7 OpAICPUDef。

约束说明

无

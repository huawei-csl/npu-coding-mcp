功能说明

在SIMT VF实现代码中调用此接口会中断算子的运行，适用于Kernel侧异常场景的调试。

函数原型

```cpp
__simt_callee__ inline void __trap()
```

参数说明

无

返回值说明

无

约束说明

无

需要包含的头文件

使用该接口需要包含"utils/debug/asc_assert.h"头文件。

```cpp
#include "utils/debug/asc_assert.h"
```

调用示例

```cpp
__simt_vf__ __launch_bounds__(1024) inline void SimtKernel(__gm__ bool* dst, __gm__ float* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;
    if (isnan(x[idx])) {        __trap();    }
```

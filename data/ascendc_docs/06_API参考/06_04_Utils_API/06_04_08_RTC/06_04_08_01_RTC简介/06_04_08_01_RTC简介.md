## 6.4.8 RTC

## 6.4.8.1 RTC 简介

RTC是Ascend C运行时编译库，通过aclrtc接口，在程序运行时，将中间代码动态编译成目标机器码，提升程序运行性能。开发指导请参考2.3.1.5 RTC。

使用aclrtc接口前需要包含如下头文件：

```cpp
#include "acl/acl_rt_compile.h"
```

需要链接的库文件：libacl_rtc.so。

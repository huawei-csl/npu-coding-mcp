功能说明

判断是否所有活跃线程的输入均不为0。

当Warp内所有活跃线程执行本接口后，对所有活跃线程的输入操作数predicate进行判断，所有活跃线程的predicate均不为0，返回1，否则返回0。Warp内所有活跃线程返回相同的结果。

函数原型

```cpp
inline int32_t asc_all(int32_t predicate)
```

参数说明

表6-1621参数说明

参数名输入/输出描述

predicate

输入操作数。

返回值说明

当Warp内所有活跃线程的输入均不为0，返回1，否则返回0。

约束说明

无

需要包含的头文件

使用该接口需要包含"simt_api/device_warp_functions.h"头文件。

```cpp
#include "simt_api/device_warp_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelAll(__gm__ int32_t* dst){    // asc_vf_call参数：dim3{1024, 1, 1}    int idx = threadIdx.x + blockIdx.x * blockDim.x;    int32_t laneId = idx % 32;    int32_t result = asc_all(laneId); // 返回值为0    dst[idx] = result;}

返回值说明

由两个half类型数字构成的新half2向量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel_make_half2(__gm__ half2* dst, __gm__ half* x, __gm__ half* y){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = make_half2(x[idx], y[idx]);}

返回值说明

由四个unsigned short类型数字构成的新ushort4向量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/vector_functions.h"头文件。

```cpp
#include "simt_api/vector_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel_make_ushort4(__gm__ ushort4* dst, __gm__ unsigned short* x, __gm__ unsigned short* y, __gm__ unsigned short* z, __gm__ unsigned short* w){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = make_ushort4(x[idx], y[idx], z[idx], w[idx]);}

功能说明

使用给定的两个unsigned long int类型的数据创建一个ulong2类型的向量。

函数原型

```cpp
inline ulong2 make_ulong2(unsigned long int x, unsigned long int y)
```

参数说明

表6-1895参数说明

参数名输入/输出

描述

x输入源操作数。

y输入源操作数。

返回值说明

由两个unsigned long int类型数字构成的新ulong2向量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/vector_functions.h"头文件。

```cpp
#include "simt_api/vector_functions.h"
```

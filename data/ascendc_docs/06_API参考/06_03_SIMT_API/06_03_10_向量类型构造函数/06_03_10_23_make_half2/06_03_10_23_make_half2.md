返回值说明

由四个signed char类型数字构成的新char4向量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/vector_functions.h"头文件。

```cpp
#include "simt_api/vector_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel_make_char4(__gm__ char4* dst, __gm__ signed char* x, __gm__ signed char* y, __gm__ signed char* z, __gm__ signed char* w){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = make_char4(x[idx], y[idx], z[idx], w[idx]);}

功能说明

使用给定的两个half类型的数据创建一个half2类型的向量。对于16位类型的数据，底层指令支持一次完成两个数据的计算，通过构造向量类型可以提高数据计算效率。

函数原型

```cpp
inline half2 make_half2(half x, half y)
```

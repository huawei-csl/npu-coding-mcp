功能说明

使用给定的两个signed char类型的数据创建一个char2类型的向量。

函数原型

```cpp
inline char2 make_char2(signed char x, signed char y)
```

参数说明

表6-1907参数说明

参数名输入/输出

描述

x输入源操作数。

y输入源操作数。

返回值说明

由两个signed char类型数字构成的新char2向量。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/vector_functions.h"头文件。

```cpp
#include "simt_api/vector_functions.h"
```

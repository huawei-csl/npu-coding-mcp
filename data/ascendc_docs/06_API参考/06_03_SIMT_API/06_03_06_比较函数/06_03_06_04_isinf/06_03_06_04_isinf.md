功能说明

判断浮点数是否为无穷。

函数原型

```cpp
inline bool isinf(float x)
```

参数说明

表6-1597参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

●false：输入不是无穷。

●true：输入为inf、-inf。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

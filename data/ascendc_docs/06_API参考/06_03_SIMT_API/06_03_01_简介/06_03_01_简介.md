## 6.3 SIMT API

## 6.3.1 简介

说明

**SIMT API当前仅支持SIMD与SIMT混合编程，SIMT编程场景待正式发布支持，请关注后续版本。**

SIMT API是面向AI处理器的并行计算编程接口，基于SIMT编程模型实现高效的数据并行计算，具体API分类请参考表6-1451。在SIMT编程中，每32个线程被分为一组，这一组线程称为Warp。Warp level接口负责处理单个Warp内32个线程的数据操作。Warp中的每个线程称为Lane，每个线程在Warp内的编号为LaneId，LaneId的范围是[0, 31]。

同步函数内存管理与同步接口，解决不同核内的线程间可能存在的数据竞争以及线程的同步问题。

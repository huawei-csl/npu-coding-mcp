需要包含的头文件

使用half类型接口需要包含"simt_api/asc_fp16.h"头文件，使用bfloat16_t类型接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"#include "simt_api/asc_bf16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelIsFinite(__gm__ half* dst, __gm__ half* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = hfloor(x[idx]);}

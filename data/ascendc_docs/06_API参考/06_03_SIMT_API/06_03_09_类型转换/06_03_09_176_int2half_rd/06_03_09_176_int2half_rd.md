参数名输入/输出

描述

x输入源操作数。

返回值说明

饱和模式下，遵循CAST_TRUNC模式，将输入int32数据转换成的half数据。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"
```

调用示例

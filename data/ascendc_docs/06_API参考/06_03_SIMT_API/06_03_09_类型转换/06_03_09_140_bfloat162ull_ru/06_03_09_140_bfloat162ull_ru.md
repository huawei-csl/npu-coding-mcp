功能说明

遵循CAST_CEIL模式，将bfloat16类型数据转换为64位无符号整数，返回转换后的值。

函数原型

```cpp
inline unsigned long long int __bfloat162ull_ru(const bfloat16_t x)
```

返回值说明

输入遵循CAST_CEIL模式转换成的64位无符号整数。特别场景说明如下：

●当x为nan时，返回值为0。

●当x为inf时，返回值为18446744073709551615。

●当x为-inf时，返回值为0。

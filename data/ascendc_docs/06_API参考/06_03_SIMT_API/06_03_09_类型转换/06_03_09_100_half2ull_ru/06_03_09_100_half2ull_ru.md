返回值说明

输入遵循CAST_FLOOR模式转换成的64位无符号整数。特别场景说明如下：

●当x为nan时，返回值为0。

●当x为inf时，返回值为18446744073709551615。

●当x为-inf时，返回值为0。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void kernel__half2ull_rd(__gm__ uint64_t* dst, __gm__ half* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = __half2ull_rd(x[idx]);}

功能说明

遵循CAST_CEIL模式，将half类型数据转换为64位无符号整数，返回转换后的值。

函数原型

```cpp
inline unsigned long long int __half2ull_ru(const half x)
```

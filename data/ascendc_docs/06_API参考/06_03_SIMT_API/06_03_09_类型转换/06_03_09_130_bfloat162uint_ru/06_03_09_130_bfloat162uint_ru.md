返回值说明

输入遵循CAST_FLOOR模式转换成的无符号整数。特别场景说明如下：

●当x为nan时，返回值为0。

●当x为inf时，返回值为4294967295。

●当x为-inf时，返回值为0。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"
```

调用示例

功能说明

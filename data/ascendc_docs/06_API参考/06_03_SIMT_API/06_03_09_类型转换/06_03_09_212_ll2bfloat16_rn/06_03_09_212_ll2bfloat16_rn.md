功能说明

遵循CAST_RINT模式，将int64类型数据转换为bfloat16类型数据，返回转换后的值。

函数原型

```cpp
inline bfloat16_t __ll2bfloat16_rn(const long long int x)
```

返回值说明

输入遵循CAST_RINT模式转换成的bfloat16类型数据。

约束说明

SIMT编程场景当前不支持使用该接口。

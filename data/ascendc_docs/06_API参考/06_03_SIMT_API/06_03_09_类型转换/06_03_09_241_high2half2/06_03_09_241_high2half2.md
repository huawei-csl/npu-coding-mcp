功能说明

将输入数据的的高16位填充到half2并返回结果。

函数原型

```cpp
inline half2 __high2half2(const half2 x)
```

参数说明

表6-1876参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的的高16位填充到half2的结果。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"
```

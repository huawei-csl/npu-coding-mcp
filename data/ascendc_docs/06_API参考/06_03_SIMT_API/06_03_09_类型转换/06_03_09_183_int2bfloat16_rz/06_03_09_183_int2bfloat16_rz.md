功能说明

遵循CAST_TRUNC模式，将int32类型数据转换为bfloat16类型数据，返回转换后的值。

函数原型

```cpp
inline bfloat16_t __int2bfloat16_rz(const int x)
```

参数说明

表6-1818参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入遵循CAST_TRUNC模式转换成的bfloat16类型数据。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_bf16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"
```

功能说明

将输入的数据分别填充为half2前后两个分量，返回填充后的数据。

函数原型

```cpp
inline half2 __halves2half2(const half x, const half y)
```

参数说明

表6-1879参数说明

参数名输入/输出

描述

x输入源操作数。

y输入源操作数。

返回值说明

输入的数据分别填充为half2前后两个分量的结果。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_fp16.h"
```

功能说明

遵循CAST_CEIL模式，将int64类型数据转换为浮点数，返回转换后的值。

函数原型

```cpp
inline float __ll2float_ru(const long long int x)
```

参数说明

表6-1840参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入遵循CAST_CEIL模式转换成的浮点数。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/device_functions.h"头文件。

```cpp
#include "simt_api/device_functions.h"
```

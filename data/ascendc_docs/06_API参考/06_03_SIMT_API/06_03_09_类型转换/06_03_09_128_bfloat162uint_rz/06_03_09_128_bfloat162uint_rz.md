功能说明

遵循CAST_TRUNC模式，将bfloat16类型数据转换为无符号整数，返回转换后的值。

函数原型

```cpp
inline unsigned int __bfloat162uint_rz(const bfloat16_t x)
```

参数说明

表6-1763参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入遵循CAST_TRUNC模式转换成的无符号整数。特别场景说明如下：

●当x为nan时，返回值为0。

●当x为inf时，返回值为4294967295。

●当x为-inf时，返回值为0。

约束说明

SIMT编程场景当前不支持使用该接口。

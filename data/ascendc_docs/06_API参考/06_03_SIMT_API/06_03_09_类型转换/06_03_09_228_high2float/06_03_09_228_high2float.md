功能说明

将输入数据的高16位转换为float类型并返回结果。

函数原型

```cpp
inline float __high2float(const half2 x)inline float __high2float(const bfloat16x2_t x)
```

参数说明

表6-1863参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的高16位转换为float类型的结果。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用bfloat16x2_t接口需要包含"simt_api/asc_bf16.h"头文件，使用half2接口需要包含"simt_api/asc_fp16.h"头文件。

```cpp
#include "simt_api/asc_bf16.h"
```

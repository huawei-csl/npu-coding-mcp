功能说明

遵循CAST_ROUND模式，将half类型数据转换为64位有符号整数，返回转换后的值。

函数原型

```cpp
inline long long int __half2ll_rna(const half x)
```

参数说明

表6-1741参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入遵循CAST_ROUND模式转换成的64位有符号整数。特别场景说明如下：

●当x为nan时，返回值为0。

●当x为inf时，返回值为9223372036854775807。

●当x为-inf时，返回值为-9223372036854775808。

约束说明

SIMT编程场景当前不支持使用该接口。

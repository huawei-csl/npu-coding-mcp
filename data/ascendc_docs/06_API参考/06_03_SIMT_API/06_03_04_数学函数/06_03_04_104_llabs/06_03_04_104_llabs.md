约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

●SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelLabs(__gm__ long int* dst, __gm__ long int* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;        dst[idx] = labs(x[idx]); }

参数名输入/输出

描述

exp输出Unified Buffer、Global Memory或栈空间的地址，用于存储以2为底的指数。

返回值说明

将x转换为归一化[1/2, 1)的有符号数乘以2的积分幂时，返回该有符号数。

●当x为inf时，返回值为inf，exp为0。

●当x为-inf时，返回值为inf，exp为0。

●当x为nan时，返回值为nan，exp为0。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：

函数原型

```cpp
inline float ldexpf(float x, int exp)
```

●当x为inf时，返回值为inf。

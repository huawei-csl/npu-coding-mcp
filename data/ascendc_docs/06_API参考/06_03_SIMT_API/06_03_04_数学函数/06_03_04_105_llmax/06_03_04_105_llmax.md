功能说明

获取两个输入数据中的最大值。

![p3251_img001.png](../../images/p3251_img001.png)

函数原型

```cpp
inline long long int llmax(const long long int x, const long long int y)
```

参数说明

表6-1559参数说明

参数名输入/输出

描述

x输入源操作数。

y输入源操作数。

返回值说明

两个输入数据中的最大值。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelLlmax(__gm__ long long int* dst, __gm__ long long int* x, __gm__ long long int* y){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = llmax(x[idx], y[idx]);   }

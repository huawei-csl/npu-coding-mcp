功能说明

获取输入数据的三角函数正切值。

![p3105_img001.png](../../images/p3105_img001.png)

函数原型

```cpp
inline float tanf(float x)
```

参数说明

表6-1455参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的正切值。

●当x为inf时，返回值为nan。

●当x为-inf时，返回值为nan。

●当x为nan时，返回值为nan。

约束说明

使用本接口时，线程配置最大不超过1024，否则有栈溢出风险。

表6-1532参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

获取输入数据x的标准正态累积分布的逆函数。

●当x为0时，返回值为-inf。

●当x为nan时，返回值为nan。

●当x为1时，返回值为inf。

●当x∉[0,1]时，返回nan。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

功能说明

函数原型

```cpp
inline float j0f(float x)
```

参数说明

表6-1533参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的0阶第一类贝塞尔函数j0的值。

●当x=inf或-inf时，返回值为0。

●当x=nan时，返回值为nan。

约束说明

使用本接口时，配置的线程数不应超过256，否则有栈溢出风险。

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

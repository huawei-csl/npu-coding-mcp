参数说明

表6-1527参数说明

参数名输入/输出

描述

x输入源操作数。

返回值说明

输入数据的伽马函数值。

●当x为+0.0时，返回值为inf。

●当x为-0.0时，返回值为-inf。

●当x为inf时，返回值为inf。

●当x为-inf时，返回值为nan。

●当x为nan时，返回值为nan。

●若返回值超出float最大范围，返回值为inf。

●若返回值超出float最小范围，返回值为-inf。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelTgamma(__gm__ float* dst, __gm__ float* x){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = tgammaf(x[idx]);}

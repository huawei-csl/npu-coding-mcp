返回值说明

x^2 + y^2的平方根。

●若x^2 + y^2的平方根超出float最大范围，返回值为inf。

●若x，y任意一个为±inf，返回值为inf。

●若x，y任意一个为nan，另外不为±inf，返回值为nan。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/math_functions.h"头文件。

```cpp
#include "simt_api/math_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelHypot(__gm__ float* dst, __gm__ float* x, __gm__ float* y) {    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = hypotf(x[idx], y[idx]);}

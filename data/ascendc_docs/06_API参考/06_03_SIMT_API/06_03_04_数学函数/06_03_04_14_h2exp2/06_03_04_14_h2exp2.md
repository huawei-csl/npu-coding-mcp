功能说明

指定输入x，对x的各元素，获取2的该元素次方。

![p3123_img001.png](../../images/p3123_img001.png)

函数原型

```cpp
inline half2 h2exp2(half2 x)inline bfloat16x2_t h2exp2(bfloat16x2_t x)
```

参数说明

表6-1468参数说明

参数名输入/输出描述

x输入源操作数。

返回值说明

输入的各元素，2的该元素次方。

●当输入元素为inf时，返回值为inf。

●当输入元素为-inf时，返回值为0。

●当输入元素为nan时，返回值为nan。

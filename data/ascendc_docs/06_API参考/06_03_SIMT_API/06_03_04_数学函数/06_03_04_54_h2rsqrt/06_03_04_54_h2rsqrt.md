功能说明

获取输入数据x各元素的平方根的倒数。

![p3178_img001.png](../../images/p3178_img001.png)

函数原型

```cpp
inline half2 h2rsqrt(half2 x)inline bfloat16x2_t h2rsqrt(bfloat16x2_t x)
```

参数说明

表6-1508参数说明

参数名输入/输出

描述

x输入源操作数，输入数据。

返回值说明

输入数据各元素的平方根的倒数。

●当输入元素为0时，返回值为inf。

●当输入元素为inf时，返回值为0。

●当输入元素为-inf时，返回值为nan。

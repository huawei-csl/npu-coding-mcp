函数原型

```cpp
unsigned int __umulhi(unsigned int x, unsigned int y)
```

参数说明

表6-1568参数说明

参数名输入/输出

描述

x输入源操作数，乘数。

y输入源操作数，乘数。

返回值说明

输入uint32类型数据x和y乘积的高32位。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/device_functions.h"头文件。

```cpp
#include "simt_api/device_functions.h"
```

调用示例

SIMD与SIMT混合编程场景：__simt_vf__ __launch_bounds__(1024) inline void KernelUmulhi(__gm__ unsigned int* dst, __gm__ unsigned int* x, __gm__ unsigned int* y){    int idx = threadIdx.x + blockIdx.x * blockDim.x;    dst[idx] = __umulhi(x[idx], y[idx]);}

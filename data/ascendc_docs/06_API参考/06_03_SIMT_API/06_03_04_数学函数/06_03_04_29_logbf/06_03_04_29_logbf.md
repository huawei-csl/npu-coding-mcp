功能说明

计算以2为底，输入数据的对数，并对结果向下取整，返回浮点数。

![p3143_img001.png](../../images/p3143_img001.png)

函数原型

```cpp
inline float logbf(float x)
```

参数说明

表6-1483参数说明

参数名输入/输出描述

x输入源操作数。

返回值说明

以2为底的x的对数，并向下取整后的值。

●当x为inf时，返回值为inf。

●当x为-inf时，返回值为inf。

●当x为nan时，返回值为nan。

●当x为0时，返回值为-inf。

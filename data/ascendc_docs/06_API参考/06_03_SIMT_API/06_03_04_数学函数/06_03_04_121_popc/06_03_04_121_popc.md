返回值说明

数据二进制低位开始的第一个比特位为1的位置。

●当x=0时，返回0。

●当x=1时，返回1。

约束说明

SIMT编程场景当前不支持使用该接口。

需要包含的头文件

使用该接口需要包含"simt_api/device_functions.h"头文件。

```cpp
#include "simt_api/device_functions.h"
```

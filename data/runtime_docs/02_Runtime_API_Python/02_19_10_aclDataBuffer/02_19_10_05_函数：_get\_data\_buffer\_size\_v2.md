获取 aclDataBuffer 类型中数据的内存大小，单位 Byte 。

- C 函数原型

size\_t aclGetDataBufferSizeV2(const aclDataBuffer *dataBuffer)

- python 函数

size = acl.get\_data\_buffer\_size\_v2(data\_buffer)

| 参数名         | 说明                                                                                 |
|-------------|------------------------------------------------------------------------------------|
| data_buffer | int ， aclDataBuffer 的指针地址。 需提前调用 acl.create_data_buffer 接口创建 aclDataBuffer 类型 的数据。 |

## 返回值说明

| 返回值   | 说明                              |
|-------|---------------------------------|
| size  | int ， aclDataBuffer 类型中数据的内存大小。 |

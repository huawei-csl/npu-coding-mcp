ret = acl.prof.mark\_ex(msg, stream)

| 参数名    | 说明               |
|--------|------------------|
| msg    | str ，打点信息字符串。    |

## 约束说明

msproftx 用于记录事件发生的时间跨度的开始时间。

调用此接口后， Profiling 自动在 Stamp 中记录开始的时间戳，将 Event type 设置为 Push/Pop 。

- C 函数原型

aclError aclprofPush(void *stamp)

- python 函数

ret = acl.prof.push(stamp)

| 参数名   | 说明                                                                  |
|-------|---------------------------------------------------------------------|
| stamp | int ， Stamp 指针地址，指代 msproftx 事件标记。指定 acl.prof.create_stamp 接口的指针地址。 |

| 返回值   | 说明                                  |
|-------|-------------------------------------|
| ret   | int ，错误码。 ● 返回 0 表示成功。 ● 返回其它值表示失败。 |

- 与 2.16.2.7 函数： pop 接口成对使用，表示时间跨度的开始和结束。
- 不能跨线程调用，若需要跨线程可使用 2.16.2.8 函数： range\_start / 2.16.2.9 函 数： range\_stop 接口。
- 在 2.16.2.2 函数： create\_stamp 接口和 2.16.2.10 函数： destroy\_stamp 接口之 间调用。

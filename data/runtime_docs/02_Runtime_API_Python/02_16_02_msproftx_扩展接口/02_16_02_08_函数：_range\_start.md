msproftx 用于记录事件发生的时间跨度的开始时间。

调用此接口后， Profiling 自动在 Stamp 记录采集开始的时间戳，将 Event type 设置为 Start/Stop ，生成一个进程唯一的 id ，并将 Stamp 保存在以进程粒度维护的一个 map 中。

## ● C 函数原型

aclError aclprofRangeStart(void *stamp, uint32\_t *rangeId)

- python 函数

range\_id, ret = acl.prof.range\_start(stamp)

| 参数名   | 说明                                                                       |
|-------|--------------------------------------------------------------------------|
| stamp | int ， Stamp 指针地址，指代 msproftx 事件标记。指定 2.16.2.2 函 数： create_stamp 接口的指针地址。 |

| 返回值      | 说明                                  |
|----------|-------------------------------------|
| range_id | int ， msproftx 事件标记的唯一标识。用于在跨线程时区分。 |

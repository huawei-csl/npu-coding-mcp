> **Section**: 2.16.2.9

msproftx 用于记录事件发生的时间跨度的结束时间。

调用此接口后， Profiling 自动在 Stamp 中记录采集结束的时间戳。

- C 函数原型 aclError aclprofRangeStop(uint32\_t *rangeId)
- python 函数

ret = acl.prof.range\_stop(range\_id)

## 参数说明

## 返回值说明

## 约束说明

| 参数名      | 说明                                  |
|----------|-------------------------------------|
| range_id | int ， msproftx 事件标记的唯一标识。用于在跨线程时区分。 |

| 返回值   | 说明                                  |
|-------|-------------------------------------|
| ret   | int ，错误码。 ● 返回 0 表示成功。 ● 返回其它值表示失败。 |

- 与 2.16.2.8 函数： range\_start 接口成对使用，表示时间跨度的开始和结束。
- 可以跨线程调用。
- 在 2.16.2.2 函数： create\_stamp 接口和 2.16.2.10 函数： destroy\_stamp 接口之 间调用。

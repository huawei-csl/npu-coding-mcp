# 总体说明

> **Section**: 2.16.2.1


- 调用接口要求： msproftx 功能相关接口须在 acl.prof.start 接口与 acl.prof.stop 接 口之间调用。其中配对使用的接口有： acl.prof.create\_stamp / acl.prof.destroy\_stamp 、 acl.prof.push / acl.prof.pop 、 acl.prof.range\_start / acl.prof.range\_stop 。
- 接口调用顺序： acl.prof.start 接口 ( 指定 Device 0 和 Device 1)--&gt; acl.prof.create\_stamp 接口 --&gt; acl.prof.set\_stamp\_trace\_message 接口 --&gt; acl.prof.mark 接口 --&gt;( acl.prof.push 接口 --&gt; acl.prof.pop 接口 ) 或 ( acl.prof.range\_start 接口 --&gt; acl.prof.range\_stop 接口 )--&gt; acl.prof.destroy\_stamp 接口 --&gt; acl.prof.stop 接口 ( 与 acl.prof.start 接口的 aclprofConfig 数据保持一致 ) 。

销毁 aclprofConfig 类型的数据，只能销毁通过 acl.prof.create\_config 接口创建的 aclprofConfig 类型。

- C 函数原型

aclError aclprofDestroyConfig(const aclprofConfig *profilerConfig)

- python 函数

ret = acl.prof.destroy\_config(profiler\_config)

| 参数名             | 说明                               |
|-----------------|----------------------------------|
| profiler_config | int ，待销毁的 aclprofConfig 类型的指针地址。 |

| 返回值   | 说明                                  |
|-------|-------------------------------------|
| ret   | int ，错误码。 ● 返回 0 表示成功。 ● 返回其它值表示失败。 |

- 与 acl.prof.create\_config 接口配对使用，先调用 acl.prof.create\_config 接口再调 用 aclprofDestroyConfig 接口。
- 同一 aclprofConfig 数据重复调用 acl.prof.destroy\_config 接口，会出现重复释放内 存的报错。

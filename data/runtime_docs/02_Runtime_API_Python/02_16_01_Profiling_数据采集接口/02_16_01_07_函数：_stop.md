停止 Profiling 数据采集。

- C 函数原型

aclError aclprofStop(const aclprofConfig *profilerConfig)

- python 函数
- ret = acl.prof.stop(profiler\_config)

| 参数名             | 说明                                                                              |
|-----------------|---------------------------------------------------------------------------------|
| profiler_config | int ，指定停止 Profiling 数据采集的配置的指针地址。 与 acl.prof.start 接口中的 aclprofConfig 类型数据保持一致。 |

| 返回值   | 说明                                  |
|-------|-------------------------------------|
| ret   | int ，错误码。 ● 返回 0 表示成功。 ● 返回其它值表示失败。 |

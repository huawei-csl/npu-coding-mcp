# 函数： bytes\_to\_ptr

> **Section**: 2.4.4


| C 函数原型    | 无                                               |
|-----------|-------------------------------------------------|
| Python 函数 | ptr = acl.util.bytes_to_ptr(data)               |
| 函数功能      | 将 bytes 对象转换成为 void* 数据，可以将转换好的数据传递给 C 函 数直接使用。 |
| 输入说明      | data ： bytes 类型数据。                              |
| 返回值说明     | ptr ： int ，可以被 C 语言访问的数据。                       |

| 约束说明   | 该函数输入的生命周期需要大于输出的生命周期，否则可能会产生 未定义行为。                                               |
|--------|------------------------------------------------------------------------------------|
| 注意事项   | 修改示例如下： bytes_in = np_arr_in.tobytes() bytes_ptr = acl.util.bytes_to_ptr(bytes_in) |

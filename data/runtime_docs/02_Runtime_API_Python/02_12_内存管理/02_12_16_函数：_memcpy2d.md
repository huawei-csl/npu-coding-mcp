实现同步内存复制，主要用于矩阵数据的复制。

Atlas 推理系列加速模块产品不支持该接口。

## ● C 函数原型

aclError aclrtMemcpy2d(void *dst, size\_t dpitch, const void *src, size\_t spitch, size\_t width, size\_t height, aclrtMemcpyKind kind)

## ● python 函数

ret = acl.rt.memcpy2d(dst, dpitch, src, spitch, width, height, kind)

| 参数名    | 说明                     |
|--------|------------------------|
| dst    | int ，目的内存地址的指针地址。      |
| dpitch | int ，目的内存中相邻两列向量的地址距离。 |
| src    | int ，源内存地址的指针地址。       |
| spitch | int ，源内存中相邻两列向量的地址距离。  |
| width  | int ，待复制的矩阵宽度。         |
| height | int ，待复制的矩阵高度。         |
| kind   | int ，内存复制的类型。          |

## 返回值说明

## 约束说明

| 返回值   | 说明                            |
|-------|-------------------------------|
| ret   | int ，错误码，返回 0 表示成功，返回其它值表示失败。 |

- 当前仅支持' ACL\_MEMCPY\_HOST\_TO\_DEVICE '类型和 ' ACL\_MEMCPY\_DEVICE\_TO\_HOST '类型的内存复制。
- Atlas 200I/500 A2 推理产品， Ascend RC 形态下，不支持调用本接口。

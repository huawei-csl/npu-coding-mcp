## 约束说明

- C 函数原型

aclError aclrtFree(void *devPtr)

- python 函数
- ret= acl.rt.free(dev\_ptr)
- acl.rt.free 接口只能释放通过 acl.rt.malloc 接口或 acl.rt.malloc\_cached 接口。
- 本接口会立刻释放传入的内存，函数内部不会进行隐式的 Device 同步或流同步。 用户需要确保调用完本接口后不再对该内存指针进行访问。

| 参数名     | 说明               |
|---------|------------------|
| dev_ptr | int ，待释放内存的指针地址。 |

| 返回值   | 说明                            |
|-------|-------------------------------|
| ret   | int ，错误码，返回 0 表示成功，返回其它值表示失败。 |

初始化内存，将内存中的内容设置为指定的值。

待初始化的内存支持在 Host 侧或 Device 侧，系统根据地址判定是 Host 还是 Device 。如 果 Host 内存不是用 acl 接口（例如 acl.rt.malloc\_host ）申请的，将会导致未定义的行 为。

- C 函数原型

aclError aclrtMemset(void *devPtr, size\_t maxCount, int32\_t value, size\_t count)

- python 函数

ret = acl.rt.memset(dev\_ptr, max\_count, value, count)

| 参数名       | 说明                           |
|-----------|------------------------------|
| dev_ptr   | int ，内存的起始地址。                |
| max_count | int ，内存的最大长度，单位 Byte 。       |
| value     | int ，设置的值。                   |
| count     | int ，需要设置为指定值的内存长度，单位 Byte 。 |

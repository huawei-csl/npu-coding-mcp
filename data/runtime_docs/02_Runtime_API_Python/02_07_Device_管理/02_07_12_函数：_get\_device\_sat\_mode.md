# 函数： get\_device\_sat\_mode

> **Section**: 2.7.12


## 产品支持情况

| 产品                     | 是否支持   |
|------------------------|--------|
| Atlas 训练系列产品           | x      |
| Atlas 推理系列产品           | x      |
| Atlas 200I/500 A2 推理产品 | x      |

设置当前 Device 的浮点计算结果输出模式。

调用该接口成功后，后续在该 Device 上新创建的 Stream 按设置的模式生效，对之前已 创建的 Stream 不生效。

- C 函数原型

aclError aclrtSetDeviceSatMode(aclrtFloatOverflowMode mode)

- python 函数

ret = acl.rt.set\_device\_sat\_mode(mode)

| 参数名   | 说明                                                       |
|-------|----------------------------------------------------------|
| mode  | int ，设置浮点计算结果输出模式，具体请参见 2.19.26 aclrtFloatOverflowMode 。 |

| 返回值   | 说明                            |
|-------|-------------------------------|
| ret   | int ，错误码，返回 0 表示成功，返回其它值表示失败。 |

| 产品            | 是否支持   |
|---------------|--------|
| Atlas 350 加速卡 | √      |

获取当前 昇 腾 AI 软件栈的运行模式。

- C 函数原型

aclError aclrtGetRunMode(aclrtRunMode *runMode)

- python 函数
- run\_mode, ret = acl.rt.get\_run\_mode()

| 参数名      | 说明                                 |
|----------|------------------------------------|
| run_mode | int ，表示运行模式。类型定义请参见 aclrtRunMode 。 |

| 返回值   | 说明                            |
|-------|-------------------------------|
| ret   | int ，错误码，返回 0 表示成功，返回其它值表示失败。 |

接口调用流程与示例，请参见运行时资源申请与释放。

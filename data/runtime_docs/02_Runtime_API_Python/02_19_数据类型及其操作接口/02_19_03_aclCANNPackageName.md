# aclCANNPackageName

> **Section**: 2.19.3


| 数据格式                  | 说明                         |
|-----------------------|----------------------------|
| ACL_PKG_NAME_CANN     | 表示查询 CANN 的版本号。            |
| ACL_PKG_NAME_RUNTIME  | 表示查询 CANN 中 Runtime 的版本号。  |
| ACL_PKG_NAME_COMPILER | 表示查询 CANN 中 Compiler 的版本号。 |
| ACL_PKG_NAME_HCCL     | 表示查询 CANN 中 HCCL 的版本号。     |
| ACL_PKG_NAME_TOOLKIT  | 表示查询 CANN 中 Toolkit 的版本号。  |

| 数据格式                    | 说明                        |
|-------------------------|---------------------------|
| ACL_PKG_NAME_OPP        | 表示查询 CANN 中 Opp 的版本号。     |
| ACL_PKG_NAME_OPP_KERNEL | 表示查询 CANN 中 kernels 的版本号。 |
| ACL_PKG_NAME_DRIVER     | 表示查询 Driver 的版本号。         |

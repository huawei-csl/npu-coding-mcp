# aclrtUtilizationInfo

> **Section**: 2.19.49


| 数据格式               | 说明                 |
|--------------------|--------------------|
| cube_utilization   | 整型， Cube 算子使用率。    |
| vector_utilization | 整型， Vector 算子使用率。  |
| aicpu_utilization  | 整型， AI CPU 算子使用率。  |
| memory_utilization | 整型， Device 内存使用率。  |
| utilization_extend | 整型，预留扩展字段，当前版本不支持。 |

# aclCANNPackageVersion

> **Section**: 2.19.4


| 成员名称           | 说明                          |
|----------------|-----------------------------|
| version        | string ，完整版本号。              |
| majorVersion   | string ，主版本号。               |
| minorVersion   | string ，次版本号                |
| releaseVersion | string ，发布号，如果查询不到，就补 0 。   |
| patchVersion   | string ，补丁版本号，如果查询不到，就补 0 。 |
| reserved       | string ，预留值，默认为空。           |

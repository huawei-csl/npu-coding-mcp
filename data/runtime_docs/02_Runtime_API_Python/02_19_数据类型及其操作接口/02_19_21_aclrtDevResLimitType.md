# aclrtDevResLimitType

> **Section**: 2.19.21


| 数据格式                          | 说明                  |
|-------------------------------|---------------------|
| ACL_RT_DEV_RES_CUBE_CORE = 0  | AI Core 或 Cube Core |
| ACL_RT_DEV_RES_VECTOR_COR E=1 | Vector Core         |

关于 Core 的定义及详细说明，请参见 aclrtDevAttr 。

- 对于以下产品， ACL\_RT\_DEV\_RES\_CUBE\_CORE 表示 AI Core 。

Atlas 训练系列产品

Atlas 推理系列产品

- 对于以下产品， ACL\_RT\_DEV\_RES\_CUBE\_CORE 表示 Cube Core 。

Atlas 350 加速卡

Atlas A3 训练系列产品 /Atlas A3 推理系列产品

Atlas A2 训练系列产品 /Atlas A2 推理系列产品

Atlas 200I/500 A2 推理产品

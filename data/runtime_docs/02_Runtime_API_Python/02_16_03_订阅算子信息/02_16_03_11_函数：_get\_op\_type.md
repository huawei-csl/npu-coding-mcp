## 功能说明

## ● python 函数

opNameLen, ret = acl.prof.get\_op\_name\_len(opInfo, opInfoLen, index)

| 参数名       | 说明                                                                                          |
|-----------|---------------------------------------------------------------------------------------------|
| opInfo    | int ，包含算子信息的地址。                                                                             |
| opInfoLen | int ，算子信息的长度。                                                                               |
| index     | int ，指定获取第几个算子的算子名称长度。 用户调用 acl.prof.get_op_num 接口获取算子数量后，这个 index 的取值范围： [0, ( 算子数量 -1)] 。 |

| 产品                                | 是否支持   |
|-----------------------------------|--------|
| Atlas 350 加速卡                     | √      |
| Atlas A3 训练系列产品 /Atlas A3 推理系 列产品 | √      |
| Atlas A2 训练系列产品 /Atlas A2 推理系 列产品 | √      |
| Atlas 训练系列产品                      | √      |
| Atlas 推理系列产品                      | √      |
| Atlas 200I/500 A2 推理产品            | √      |

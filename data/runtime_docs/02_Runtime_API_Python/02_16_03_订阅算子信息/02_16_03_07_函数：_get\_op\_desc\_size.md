获取单个算子数据结构的大小，单位为 Byte 。当前版本中约定每个算子数据结构的大 小是一样的。

建议用户新建一个线程，在新线程内调用该接口，否则可能阻塞主线程中的其它任务 调度。

- C 函数原型

aclError aclprofGetOpDescSize(size\_t *opDescSize)

- python 函数

op\_desc\_size, ret = acl.prof.get\_op\_desc\_size()

无

| 返回值          | 说明                                  |
|--------------|-------------------------------------|
| op_desc_size | int ，算子数据结构的大小。                     |
| ret          | int ，错误码。 ● 返回 0 表示成功。 ● 返回其它值表示失败。 |

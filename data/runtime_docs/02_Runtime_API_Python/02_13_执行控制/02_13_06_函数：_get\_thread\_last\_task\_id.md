获取当前线程的最后一个下发的 Task ID 。

## 函数原型

## 参数说明

## 返回值说明

| 返回值    | 说明                                                                                   |
|--------|--------------------------------------------------------------------------------------|
| taskId | unsigned int ， Task ID 。当前线程的最后一个下发的 Task ID 。此处的 Task 表示由用户显式创建的 Stream 上下发的 Task 。 |
| ret    | int ，错误码，返回 0 表示成功，返回其它值表示失败。                                                        |

用户销毁 Allocator 前，需调用本接口取消注册用户提供的 Allocator 以及 Allocator 对应 的回调函数。

待取消注册的 Stream 不存在，或多次调用本接口取消注册，本接口内部不做任何操 作，返回成功。

aclError aclrtAllocatorUnregister(aclrtStream stream)

| 参数名    | 输入 / 输 出   | 说明                                            |
|--------|------------|-----------------------------------------------|
| stream | 输入         | 该 Allocator 对应的 stream 。类型定义请参见 aclrtStream 。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

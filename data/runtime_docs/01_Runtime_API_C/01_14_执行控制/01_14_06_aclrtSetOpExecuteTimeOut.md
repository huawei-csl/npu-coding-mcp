## 约束说明

设置算子执行的超时时间，单位为秒。一个进程内多次调用本接口，则以最后一次设 置的时间为准。

对于以下产品型号，建议使用 aclrtSetOpExecuteTimeOutV2 接口，该接口会返回实际 生效的超时时间：

Atlas 350 加速卡

Atlas A3 训练系列产品 /Atlas A3 推理系列产品

Atlas A2 训练系列产品 /Atlas A2 推理系列产品

aclError aclrtSetOpExecuteTimeOut(uint32\_t timeout)

| 参数名     | 输入 / 输 出   | 说明                                   |
|---------|------------|--------------------------------------|
| timeout | 输入         | 设置超时时间，单位为秒。 将该参数设置为 0 时，表示使用最大超时时间。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

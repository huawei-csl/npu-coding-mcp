获取当前线程的最后一个下发的 Task ID 。

aclError aclrtGetThreadLastTaskId(uint32\_t *taskId)

| 参数名    | 输入 / 输 出   | 说明                                                            |
|--------|------------|---------------------------------------------------------------|
| taskId | 输出         | 当前线程的最后一个下发的 Task ID 。 此处的 Task 表示由用户显式创建的 Stream 上下发的 Task 。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

设置算子执行的超时时间，单位为毫秒。

aclError aclrtSetOpExecuteTimeOutWithMs(uint32\_t timeout)

| 参数名     | 输入 / 输 出   | 说明                                    |
|---------|------------|---------------------------------------|
| timeout | 输入         | 设置超时时间，单位为毫秒。将该参数设置为 0 时，表示 使用最大超时时间。 |

## 返回值说明

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

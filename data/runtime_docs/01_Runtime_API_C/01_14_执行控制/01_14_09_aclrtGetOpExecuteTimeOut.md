获取 AI Core 算子执行的超时时间。

aclError aclrtGetOpExecuteTimeout(uint32\_t *const timeoutMs)

| 参数名        | 输入 / 输 出   | 说明                                                                                                |
|------------|------------|---------------------------------------------------------------------------------------------------|
| timeout Ms | 输出         | 超时时间，单位为毫秒。 若已调用 set 接口（例如 aclrtSetOpExecuteTimeOut ）设置 过超时时间，则返回硬件的实际超时时间，否则，返回 AI Core 的默认超时时间。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

设置算子执行的超时时间，单位为微秒。

aclError aclrtSetOpExecuteTimeOutV2(uint64\_t timeout,  uint64\_t *actualTimeout)

| 参数名     | 输入 / 输 出   | 说明                                    |
|---------|------------|---------------------------------------|
| timeout | 输入         | 设置超时时间，单位为微秒。 将该参数设置为 0 时，表示使用最大超时时间。 |

## 返回值说明

| 参数名            | 输入 / 输 出   | 说明                                                                                                      |
|----------------|------------|---------------------------------------------------------------------------------------------------------|
| actualTimeo ut | 输出         | 返回实际生效的超时时间，单位为微秒。 对于以下产品型号，此处只能获取 AI Core 算子执行的超 时时间： Atlas 200I/500 A2 推理产品 Atlas 推理系列产品 Atlas 训练系列产品 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

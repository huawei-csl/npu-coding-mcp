创建 Event ，创建出来的 Event 可用于统计两个 Event 之间的耗时、多 Stream 之间的任务 同步等场景。

aclError aclrtCreateEvent(aclrtEvent *event)

| 参数名   | 输入 / 输 出   | 说明                             |
|-------|------------|--------------------------------|
| event | 输出         | Event 的指针。类型定义请参见 aclrtEvent 。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

采用本 API 创建的 Event 不支持在 aclrtResetEvent 接口中使用，否则会导致未定义的行 为。

调用本接口创建 Event 时，并不会实际申请 Event 资源，只有在调用 aclrtRecordEvent 接口时，才会进行资源申请，因此在调用 aclrtRecordEvent 时，可能会出现线程阻 塞，等待 Event 资源的释放。

## 功能说明

获取算子类型的字符串长度，用于内存申请。

建议用户新建一个线程，在新线程内调用该接口，否则可能阻塞主线程中的其它任务 调度。

aclError aclprofGetOpTypeLen(const void *opInfo, size\_t opInfoLen, uint32\_t index, size\_t *opTypeLen)

| 参数名       | 输入 / 输 出   | 说明                                                                                  |
|-----------|------------|-------------------------------------------------------------------------------------|
| opInfo    | 输入         | 包含算子信息的地址。                                                                          |
| opInfoLen | 输入         | 算子信息的长度。                                                                            |
| index     | 输入         | 指定获取第几个算子的算子类型名称。 用户调用 aclprofGetOpNum 接口获取算子数量后，这 个 index 的取值范围： [0, ( 算子数量 -1)] 。 |
| opTypeLen | 输出         | opType 的长度。                                                                         |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

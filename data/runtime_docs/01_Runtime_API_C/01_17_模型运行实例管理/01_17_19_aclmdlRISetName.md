## 返回值说明

设置模型运行实例的名称。若针对同一个模型运行实例设置多次，以最后一次为准。

aclError aclmdlRISetName(aclmdlRI modelRI, const char *name)

| 参数名     | 输入 / 输 出   | 说明                                                                                                                                                                                                          |
|---------|------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| modelRI | 输入         | 模型运行实例。类型定义请参见 aclmdlRI 。 modelRI 支持通过以下方式获取： ● 调用 aclmdlRICaptureBegin 接口捕获 Stream 上下发的 任务后，可通过 aclmdlRICaptureGetInfo 接口获取模型 运行实例，再传入本接口。 ● 调用 aclmdlRIBuildBegin 、 aclmdlRIBuildEnd 等接口 构建模型运行实例，再传入本接口。 |
| name    | 输入         | 名称。                                                                                                                                                                                                         |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

执行模型。异步接口。

aclError aclmdlRIExecuteAsync(aclmdlRI modelRI, aclrtStream stream)

| 参数名     | 输入 / 输 出   | 说明                                                                                                                                                                                                          |
|---------|------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| modelRI | 输入         | 模型运行实例。类型定义请参见 aclmdlRI 。 modelRI 支持通过以下方式获取： ● 调用 aclmdlRICaptureBegin 接口捕获 Stream 上下发 的任务后，可通过 aclmdlRICaptureGetInfo 接口获 取模型运行实例，再传入本接口。 ● 调用 aclmdlRIBuildBegin 、 aclmdlRIBuildEnd 等接 口构建模型运行实例，再传入本接口。 |
| stream  | 输入         | 指定 Stream ，用于执行模型推理任务。类型定义请参见 aclrtStream 。                                                                                                                                                                 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

接口调用示例，参见单流捕获。

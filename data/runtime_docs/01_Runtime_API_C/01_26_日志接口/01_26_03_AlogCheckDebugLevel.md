## 函数功能

| 参数      | 输入 / 输出   | 说明                                                                                       |
|---------|-----------|------------------------------------------------------------------------------------------|
| logType | 输入        | 本条日志类型，枚举值请参见 1.26.4 数据类型定义。传入其他值 视作调试日志。                                                |
| level   | 输入        | 本条日志级别，宏定义请参见 1.26.4 数据类型定义。 说明 运行日志不会记录 debug 级别日志， DLOG_TYPE_RUN 和 DLOG_DEBUG 不可 组合使用。 |
| fmt     | 输入        | 待打印的内容。 ● 接口不校验内容合法性。 ● 单条日志最大长度为 1024 字节， 超出则会截断。                                       |

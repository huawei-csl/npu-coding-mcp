创建 Notify 。

aclError aclrtCreateNotify(aclrtNotify *notify, uint64\_t flag)

| 参数名    | 输入 / 输出   | 说明                                                                                                                                                                                                                                                             |
|--------|-----------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| notify | 输出        | Notify 的指针。类型定义请参见 aclrtNotify 。                                                                                                                                                                                                                               |
| flag   | 输入        | Notify 指针的fl ag 。 当前支持将fl ag 设置为如下宏： ● ACL_NOTIFY_DEFAULT ：使能该 bit 表示创建的 Notify 默认在 Host 上调用。 ● ACL_NOTIFY_DEVICE_USE_ONLY ：使能该 bit 表示创建 的 Notify 仅在 Device 上调用。 宏的定义如下： #define ACL_NOTIFY_DEFAULT 0x00000000U #define ACL_NOTIFY_DEVICE_USE_ONLY 0x00000001U |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

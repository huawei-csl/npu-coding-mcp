|----------------------------------|--------|
| Atlas 350 加速卡                    | √      |

| Atlas 200I/500 A2 推理产品           | ☓      |
| Atlas 推理系列产品                     | √      |
| Atlas 训练系列产品                     | √      |

在本进程中获取 key 的信息，并返回本进程可以使用的 Notify 指针。

本接口需与其它接口配合使用，以便实现多 Device 上不同进程间的任务同步，请参见 1.10.7 aclrtNotifyGetExportKey 接口处的说明。

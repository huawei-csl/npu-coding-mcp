# 内存使用说明

> **Section**: 1.13.1


各产品型号在内存使用上有一些注意事项，如下所示。

| 型号     | 注意事项                                                                                                                                                                                                                                                                                                      |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 各型号都涉及 | ● 若涉及媒体数据处理功能，由于媒体数据处理功能 对存放输入、输出数据的内存有更高的要求，因此 需调用专用的内存申请接口，例如 hi_mpi_dvpp_malloc 。 ● 调用专用的内存申请接口申请出来的内存可以满足 媒体数据处理的要求，也可以在其它任务中使用， 例如，从性能角度，为了减少拷贝，媒体数据处理 的输出作为模型推理的输入，实现内存复用。 ● 但由于媒体数据处理访问的地址空间有限，为确保 媒体数据处理时内存足够，除媒体数据处理功能外 的其它功能（例如，模型加载），建议调用内存管 理下的接口申请内存，例如 aclrtMalloc 接口、 aclrtMallocHost 接口等。 |

取消注册 Host 内存。

本接口与 aclrtHostRegister 接口成对使用。

aclError aclrtHostUnregister(void *ptr)

| 参数名   | 输入 / 输 出   | 说明          |
|-------|------------|-------------|
| ptr   | 输入         | Host 侧内存地址。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

获取当前线程的 Runtime （运行时管理模块）错误码，获取后清空当前线程的错误 码，这时在线程中无新增错误码之前，调用本接口获取到的是 ACL\_SUCCESS 。

aclError aclrtGetLastError(aclrtLastErrLevel level)

| 参数名   | 输入 / 输 出   | 说明                                                |
|-------|------------|---------------------------------------------------|
| level | 输入         | 指定获取错误码的级别，当前仅支持线程级别。类型定 义请参见 aclrtLastErrLevel 。 |

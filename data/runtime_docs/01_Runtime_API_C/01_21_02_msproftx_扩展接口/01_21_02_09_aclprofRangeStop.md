msproftx 用于记录事件发生的时间跨度的结束时间。

调用此接口后， Profiling 自动在 Stamp 指针中记录采集结束的时间戳。

aclError aclprofRangeStop(uint32\_t rangeId)

| 参数名     | 输入 / 输 出   | 说明                            |
|---------|------------|-------------------------------|
| rangeId | 输出         | msproftx 事件标记的唯一标识。用于在跨线程时区分。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

- 与 1.21.2.8 aclprofRangeStart 接口成对使用，表示时间跨度的开始和结束。
- 在 1.21.2.2 aclprofCreateStamp 接口和 1.21.2.10 aclprofDestroyStamp 接口之 间调用。
- 可以跨线程调用。

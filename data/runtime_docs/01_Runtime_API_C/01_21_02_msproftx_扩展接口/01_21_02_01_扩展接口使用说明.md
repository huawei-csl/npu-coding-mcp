# 扩展接口使用说明

> **Section**: 1.21.2.1


- 调用接口要求： msproftx 功能相关接口须在 aclprofStart 接口与 aclprofStop 接口 之间调用。其中配对使用的接口有： aclprofCreateStamp / aclprofDestroyStamp 、 aclprofPush / aclprofPop 、 aclprofRangeStart / aclprofRangeStop 。
- 接口调用顺序： aclprofStart 接口 ( 指定 Device 0 和 Device 1)--&gt; aclprofCreateStamp 接口 --&gt; aclprofSetStampTraceMessage 接口 --&gt; aclprofMark 接口 --&gt;( aclprofPush 接口 --&gt; aclprofPop 接口 ) 或 ( aclprofRangeStart 接口 --&gt; aclprofRangeStop 接口 )--&gt; aclprofDestroyStamp 接口 --&gt; aclprofStop 接口 ( 与 aclprofStart 接口的 aclprofConfig 数据保持一致 ) 。

阻塞 Host 侧当前线程直到指定 Stream 中的所有任务都完成。

aclError aclrtSynchronizeStream(aclrtStream stream)

| 参数名    | 输入 / 输 出   | 说明                                        |
|--------|------------|-------------------------------------------|
| stream | 输入         | 指定需要完成所有任务的 Stream 。类型定义请参见 aclrtStream 。 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

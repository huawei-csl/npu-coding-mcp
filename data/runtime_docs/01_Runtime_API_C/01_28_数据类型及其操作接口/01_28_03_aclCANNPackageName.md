# aclCANNPackageName

> **Section**: 1.28.3


设置为 ACL\_PKG\_NAME\_CANN ，表示查询 CANN 的版本号；设置为其它值，表示查询 CANN 中具体某个模块的版本号。

```
typedef enum aclCANNPackageName { ACL_PKG_NAME_CANN, ACL_PKG_NAME_RUNTIME, ACL_PKG_NAME_COMPILER, ACL_PKG_NAME_HCCL, ACL_PKG_NAME_TOOLKIT, ACL_PKG_NAME_OPP, ACL_PKG_NAME_OPP_KERNEL, ACL_PKG_NAME_DRIVER } aclCANNPackageName;
```

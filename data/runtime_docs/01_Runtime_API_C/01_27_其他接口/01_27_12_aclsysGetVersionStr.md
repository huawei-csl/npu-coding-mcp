根据软件包名称查询版本号，版本号是字符串类型，与各包安装目录下 version.info 文 件中的 version 保持一致。

aclError aclsysGetVersionStr(char *pkgName, char *versionStr)

| 参数名      | 输入 / 输 出   | 说明                                                                                                                                                                                     |
|----------|------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| pkgNam e | 输入         | 软件包名称。 CANN 包名称与与 ${INSTALL_DIR}/share/info 下的目录名称 保持一致。 ${INSTALL_DIR} 请替换为 CANN 软件安装后文件 存储路径。以 root 用户安装为例，安装后文件默认存储路径 为： /usr/local/Ascend/cann 。 驱动包名称为 driver 。 固件包名称为fi rmware 。 |

## 返回值说明

| 参数名         | 输入 / 输 出   | 说明                                                                                                                                                                                  |
|-------------|------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| versionSt r | 输出         | 版本号。 建议用户声明一个长度为 ACL_PKG_VERSION_MAX_SIZE 的 字符数组，用于存放版本号。例如： char versionStr[ACL_PKG_VERSION_MAX_SIZE] = {0}; ACL_PKG_VERSION_MAX_SIZE 宏的定义如下： #define ACL_PKG_VERSION_MAX_SIZE 128 |

返回 0 表示成功，返回其他值表示失败，请参见 1.28.1 aclError 。

# 预定义宏

> **Section**: 4.3


毕 昇 编译器如其他语言一样，会提供一些内置的宏方便用户编写程序。这些宏在 compiler 包中的头文件中可以看到。以下表格介绍一些常用的宏及其说明。

## 表 4-1 毕 昇 编译器内置常用的预定义宏

| 宏名           | 值                               | 说明                                                   |
|--------------|---------------------------------|------------------------------------------------------|
| __NPU_ARCH__ | 2201                            | 只在 Device 侧定义，表示 芯片架构编号， 2201 对应 A2/A3               |
| __DAV_VEC__  | 不涉及                             | 只在 Device 侧定义，表示 Vec 核                               |
| __DAV_CUBE__ | 不涉及                             | 只在 Device 侧定义，表示 Cube 核                              |
| __global__   | __attribute__((cce_kernel ))    | 在 Host 和 Device 侧都定 义，只能在 Device 侧使 用， Kernel 函数的标识符 |
| __ubuf__     | __attribute__((cce_unif_b uff)) | 在 Host 和 Device 侧都定 义，只能在 Device 侧使 用，表示 ubuf 地址空间   |

## 注意事项

| 宏名       | 值                                  | 说明                                                 |
|----------|------------------------------------|----------------------------------------------------|
| __fbuf__ | __attribute__((cce_fixpipe _buff)) | 在 Host 和 Device 侧都定 义，只能在 Device 侧使 用，表示 fbuf 地址空间 |

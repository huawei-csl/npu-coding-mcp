# Mix 算子示例

> **Section**: 5.3


编译时，须指定 CANN 包安装路径下对应的 include 目录，编译 Cube 代码，需要指定 aicore 类型为： dav-c220-cube 。

## 执行如下命令编译生成可执行文件：

```
/root/lhj/env/latest/compiler/ccec_compiler/bin/bisheng \ -O2 \ --cce-aicore-arch=dav-c220-cube \ -I ${install_path}/latest/include \ -L ${install_path}/latest/lib64 \ -I ${install_path}/include \ -lascendcl -lruntime \ main.cce \ cube_kernel.cce \ -o test
```

# vmax/vmin

> **Section**: 6.3.8.1


## 功能说明

vmax: 比较两个向量的元素，并将其中较大的值写入目标向量。

each\_element\_of([dst]) = max(each\_element\_of([src0]), each\_element\_of([src1]))

vmin: 比较两个向量的元素，并将其中较小的值写入目标向量。

each\_element\_of([dst]) = min(each\_element\_of([src0]), each\_element\_of([src1]))

上述接口均支持通过 MASK 控制哪些元素参与计算。

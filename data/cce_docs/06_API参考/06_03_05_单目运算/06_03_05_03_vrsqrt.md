# vrsqrt

> **Section**: 6.3.5.3


计算每个向量元素的算术平方根，计算公式如下：

[dst] = sqrt([src])

src 应为正值，否则会导致未知结果并引发异常。

以 block （ 32Byte ）为单位完成计算，一次完成 8 个 block 的计算。

该接口支持通过 MASK 控制哪些元素参与计算。

// 相同接口的不同原型区别在于源地址和目的地址的数据类型不同。

void vsqrt(\_\_ubuf\_\_ half *dst, \_\_ubuf\_\_ half *src, uint8\_t repeat, uint16\_t dstBlockStride, uint16\_t srcBlockStride, uint16\_t dstRepeatStride, uint16\_t srcRepeatStride);

void vsqrt(\_\_ubuf\_\_ float *dst, \_\_ubuf\_\_ float *src, uint8\_t repeat, uint16\_t dstBlockStride, uint16\_t srcBlockStride, uint16\_t dstRepeatStride, uint16\_t srcRepeatStride);

参数含义见 表 1 单目运算参数说明。

PIPE\_V

计算每个向量元素的近似算术平方根倒数，公式如下：

[dst] = 1 / sqrt([src])

src 应为正值，否则会导致未知结果并引发异常。

该接口的行为与 ARM v8.2 接口 FRSQRTE 相同。

以 block （ 32Byte ）为单位完成计算，一次完成 8 个 block 的计算。

该接口支持通过 MASK 控制哪些元素参与计算。

// 相同接口的不同原型区别在于源地址和目的地址的数据类型不同。

void vrsqrt(\_\_ubuf\_\_ half *dst, \_\_ubuf\_\_ half *src, uint8\_t repeat, uint16\_t dstBlockStride, uint16\_t srcBlockStride, uint16\_t dstRepeatStride, uint16\_t srcRepeatStride);

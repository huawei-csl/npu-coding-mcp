CCE （ Cube-based Computing Engine ） Intrinsic 接口是基于 C 语言扩展的 昇 腾硬件接 口，通过 CCE Intrinsic 可控制细粒度内存分配、数据同步、 double buffer ，实现对 昇 腾 AI 处理器极致性能挖掘。

当前 CCE Intrinsic 接口支持的产品型号：

Atlas A3 训练系列产品 /Atlas A3 推理系列产品

Atlas A2 训练系列产品 /Atlas A2 推理系列产品

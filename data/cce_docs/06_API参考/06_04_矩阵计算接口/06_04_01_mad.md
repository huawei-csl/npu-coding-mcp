# mad

> **Section**: 6.4.1


## 功能说明

## 接口原型

## 两个矩阵相乘并相加运算，计算公式如下：

c = a * b + c

矩阵 a 的大小为 m * k ，矩阵 b 的大小为 k * n ，矩阵 c 的大小为 m * n 。

如果 m, k, n 中的任何一个为零，该接口将被视为 NOP 并报错。

# 通用说明

> **Section**: 6.5.1


## 逻辑位置和物理存储的映射关系

数据的存储位置均使用逻辑位置来表达，从而隐藏了硬件架构的差异。逻辑位置类型 包括： GM 、 UBUF 、 CBUF 等，其与物理内存的映射关系如下表所示。

表 6-16 逻辑位置与物理内存映射关系表

| 逻辑位置   | 物理内存               |
|--------|--------------------|
| GM/OUT | Global Memory (GM) |

| 逻辑位置   | 物理内存                |
|--------|---------------------|
| UBUF   | Unified Buffer (UB) |
| CBUF   | L1 Buffer           |
| CA     | L0A Buffer          |
| CB     | L0B Buffer          |
| CC     | L0C Buffer          |
| BT     | Bias Table          |

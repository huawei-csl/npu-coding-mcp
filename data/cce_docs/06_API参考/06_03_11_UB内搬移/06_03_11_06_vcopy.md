# vcopy

> **Section**: 6.3.11.6


## 功能说明

UB 内数据搬运。从 src 地址搬运至 dst 地址。

该接口支持 MASK 配置。只有 MASK 中控制位为 1 的元素才会写入 dst （控制位为 0 的元 素不复制）。 b16 单次处理 src 中 128 个连续元素（包含控制位为 0 的元素）， b32 单次 处理 src 中 64 个连续元素（包含控制位为 0 的元素）。 MASK 配置详见 6.3.2.2 set\_vector\_mask 。

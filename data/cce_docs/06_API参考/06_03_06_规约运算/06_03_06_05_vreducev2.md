# vreducev2

> **Section**: 6.3.6.5


## 功能说明

把 src0 处的源向量缩减成一个更短的向量，结果存到 dst 。（如果 patternMode 等于 0 则 从 src1 中读取 mask ，否则按照 patternMode 对应的 mask 进行计算，详见表 2 patternMode 参数 mask 对照， mask 控制位 0 不参与计算，控制位 1 参与计算）。

dst ， src0 ， scr1 必须是 32B 对齐的。

该接口中 MASK 被忽略。

// 相同接口的不同原型区别在于源地址和目的地址的数据类型不同

void vreducev2(\_\_ubuf\_\_ uint16\_t *dst, \_\_ubuf\_\_ uint16\_t *src0, \_\_ubuf\_\_ uint16\_t *src1, uint16\_t repeat, uint8\_t src0BlockStride, uint8\_t patternMode, uint16\_t src0RepeatStride, uint8\_t src1RepeatStride);

void vreducev2(\_\_ubuf\_\_ uint32\_t *dst, \_\_ubuf\_\_ uint32\_t *src0, \_\_ubuf\_\_ uint32\_t *src1, uint16\_t repeat, uint8\_t src0BlockStride, uint8\_t patternMode, uint16\_t src0RepeatStride, uint8\_t src1RepeatStride);

## 表 6-9 vreducev2 特有参数说明

| 参数名          | 说明                                                                                         | 取值范 围   | 单位   |
|--------------|--------------------------------------------------------------------------------------------|---------|------|
| patternM ode | ● patternMode 为 0 时从 src1 中读取 mask 。 ● patternMode 为 1~7 是 7 种不同的 reduce 模式，详 情请参见表 6-10 。 | [0, 7]  | /    |

表 6-10 patternMode 参数 mask 对照

|   patternMode | mask 模式                       |
|---------------|-------------------------------|
|             0 | normal mode ，即从 src1 中读取 mask |
|             1 | 010101 … 01                   |
|             2 | 101010 … 10                   |
|             3 | 00010001 … 0001               |
|             4 | 00100010 … 0010               |
|             5 | 01000100 … 0100               |
|             6 | 10001000 … 1000               |
|             7 | 11111111 … 1111               |

其他参数含义见 表 1 单目运算参数说明。

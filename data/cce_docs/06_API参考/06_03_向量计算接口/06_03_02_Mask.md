# Mask

> **Section**: 6.3.2


| 参数名               | 说明                                                        | 取值范围       | 单位   |
|-------------------|-----------------------------------------------------------|------------|------|
| dstRepeat Stride  | 相邻两次执行间，目的操作数相 同 block 首地址间地址步长。 0 表 示两次迭代的所有对应块指向相 同的地址。 | [0, 2^8-1] | 32B  |
| src0Repea tStride | 相邻两次执行间，源操作数 0 相同 block 首地址间地址步长。                         | [0, 2^8-1] | 32B  |
| src1Repea tStride | 相邻两次执行间，源操作数 1 相同 block 首地址间地址步长。                         | [0, 2^8-1] | 32B  |

参数表中的 Stride 均指代相邻两个首地址间的步长。

## 共有三种硬同步模式：

- mode 0: 所有用到的 AIC 直接进行同步或所有 AIV 间进行同步。适用场景为所有核 分别计算中间结果后，由一个核进行 reduce 运算。
- mode 1: 同一个 Group 下的 AIV 间同步。
- mode 2: 同一个 Group 下的 AIC 和 AIV 间同步。多用于 AIC/AIV 分别负责算子执行中 的不同子流程（有严格顺序要求）。

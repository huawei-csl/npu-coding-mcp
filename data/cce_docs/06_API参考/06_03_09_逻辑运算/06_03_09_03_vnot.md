# vnot

> **Section**: 6.3.9.3


对每个向量元素进行按位取反运算，单次 repeat 中向量为 8*256 bits ，计算公式如下：

[dst] i = ~[src] i

该接口支持通过 MASK 控制哪些元素参与计算。

// 相同接口的不同原型区别在于源地址和目的地址的数据类型不同 void vnot(\_\_ubuf\_\_ uint16\_t *dst, \_\_ubuf\_\_ uint16\_t *src, uint8\_t repeat, uint16\_t dstBlockStride, uint16\_t srcBlockStride, uint16\_t dstRepeatStride, uint16\_t srcRepeatStride);

void vnot(\_\_ubuf\_\_ int16\_t *dst, \_\_ubuf\_\_ int16\_t *src, uint8\_t repeat, uint16\_t dstBlockStride, uint16\_t srcBlockStride, uint16\_t dstRepeatStride, uint16\_t srcRepeatStride);

参数含义见 表 1 单目运算参数说明。

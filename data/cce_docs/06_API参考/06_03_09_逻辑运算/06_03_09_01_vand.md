## 流水类型

对 src0 和 src1 的每个向量元素进行按位与运算，单次 repeat 中每个向量为 8*256 bits ， 计算公式如下：

[dst] i = [src0] i &amp; [src1] i

该接口支持通过 MASK 控制哪些元素参与计算。

// 相同接口的不同原型区别在于源地址和目的地址的数据类型不同

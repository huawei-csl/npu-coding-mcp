## 功能说明

PIPE\_V

每个元素进行 shlDistance 位数的左移操作，其中 int16 类型要求 shlDistance ≤ 16 ， int32 类型要求 shlDistance ≤ 32 ，计算公式如下：

each\_element\_of([dst]) = each\_element\_of([src]) &lt;&lt; shlDistance

以 block （ 32Byte ）为单位完成计算，一次完成 8 个 block 的计算。

该接口支持通过 MASK 控制哪些元素参与计算。

# vshr

> **Section**: 6.3.4.8


每个元素进行 shrDistance 位数的右移操作，其中 int16 和 int32 采用算术右移， uint16 和 uint32 采用逻辑右移。 int16 和 uint16 类型要求 shrDistance ≤ 16 ， int32 和 uint32 类型要求 shrDistance ≤ 32 。对于 int16 和 int32 类型语义如下：

```
if (round == 1) if (src_ele[i][shrDistance - 1] == 1) // src_ele[i][shrDistance - 1] 指的是 src_ele 向量中第 i 维元素的第 shrDistance - 1 个二进制位 dst_ele[i] = arithmetic_right_shift(src_ele[i], shrDistance) + 1 else dst_ele[i] = arithmetic_right_shift(src_ele[i], shrDistance) else: dst_ele[i] = arithmetic_right_shift(src_ele[i], shrDistance)
```

对于 uint16 和 uint32 类型， round 不生效，语义如下：

dst\_ele[i] = logic\_right\_shift(src\_ele[i], shrDistance)

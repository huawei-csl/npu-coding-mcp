# vmulconv

> **Section**: 6.3.4.11


## 功能说明

两个源均为 f16 值，将它们相乘，并附加一个 f16 转 int8 或 f16 转 uint8 操作。该接 口遵循双目向量模板，计算宽度取决于较大的数据元素。例如， f162s8 转换包含 128 个 f16 和 128 个 s8 元素，目标仅 4 个 block ，类似 vconv ，目标仅包含 4 个 block 而非 8 个，因为源已经是 8 个 block 了，转换前后元素个数不变，但数据类型位宽减半，所以 目的 block 数目为 4 。因此，填写对应的参数时，如果希望 dst 连续存放， dstRepeatStride 应该设为 4 （而不是常见的 8 ）。

舍入模式为四舍五入，就近取偶（用法类似 C 语言中的 rint ），计算公式如下：

[dst] = convert(ReLU([src0] * [src1]))

该接口支持通过 MASK 控制哪些元素参与计算。

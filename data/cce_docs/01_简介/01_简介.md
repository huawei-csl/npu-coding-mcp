# 简介

> **Section**: 1


毕 昇 编译器 CCE （ Cube-based Computing Engine ） Intrinsic 是基于 C 语言扩展的 昇 腾 硬件接口，本文重点介绍了基于 CCE Intrinsic 的异构编程与多流水并行编程，并详细说 明了毕 昇 编译器 CCE Intrinsic 关键字、变量类型、地址空间等特性，开发者可通过 CCE Intrinsic 进行细粒度内存分配、数据同步、 double buffer 等操作，实现高性能算子开 发，挖掘 昇 腾 AI 处理器极致性能。

当前 CCE Intrinsic 接口支持的产品型号：

Atlas A3 训练系列产品 /Atlas A3 推理系列产品

Atlas A2 训练系列产品 /Atlas A2 推理系列产品

关于硬件架构与数据处理原理，请参见《 Ascend C 算子开发》的'基础架构'章节。
